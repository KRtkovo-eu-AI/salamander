Open Salamander: Plugins: Network: Introduction

Getting Started with Network Plugin

With the Network plugin you can browse the network directly in the panel. It displays the list of available computers on the network and their shares.

Browsing the network with this plugin is comfortable because it is non-blocking (the panel is not blocked when reading information from the network) and cached (all visited computers and their shares are remembered).

See File System section in Using Plugins for a description of basic work with file system plugins.
