Verifying Checksums

Use this dialog to verify checksums from SFV, MD5, SHA-1, SHA-256, and SHA-512 files. You can see the result of verification in the column Status. Please note that this dialog is not modal (blocking), so you can continue in your work in Altap Salamander while verification of checksums is in progress.

To verify checksums:

  1. Focus the SFV, MD5, SHA-1, SHA-256, or SHA-512 file with checksums.
  2. Open the Verify Checksums dialog box:
    Menu:Plugins/Checksum/Verify Checksums...
    Shortcut key:Ctrl+Shift+V
  3. Wait for verification of checksums.
  4. Click the OK button to close the dialog box.
