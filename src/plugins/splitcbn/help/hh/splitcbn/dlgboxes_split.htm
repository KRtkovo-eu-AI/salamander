Open Salamander: Plugins: Split & Combine: Dialog Boxes

Split

Use this dialog box to split file to parts.

Options

Split to
Specify here the directory where you want to place splitted parts of file to. See Split & Combine Configuration for options related to generating of path to this directory.
Size of each part
Specify the maximal size of one part of splitted file. You can choose some predefined value from following combo box or you can enter own number and specify size unit (B, KB, MB, etc). Size of the last part is displayed.
Number of parts
Enter here the requested number of parts of splitted file.
Configure
Opens Split & Combine Configuration. You can select more options for splitting here.
