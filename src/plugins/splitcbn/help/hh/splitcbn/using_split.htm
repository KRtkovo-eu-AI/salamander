Splitting File to Parts

You can split file to as many parts as you need or to parts with specified maximal size (e.g. for transfer on removable media - CD/DVD/floppies).

To split file to parts:

  1. Focus the file you want to split.
  2. Open the Split dialog box:
    Menu:Plugins/Split & Combine/Split File...
  3. Specify the maximal size of one part of splitted file in the Size of each part combo box. Or enter the requested number of parts of splitted file in the Number of parts edit box.
  4. Choose the OK button.
