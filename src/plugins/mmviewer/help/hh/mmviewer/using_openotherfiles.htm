Opening Previous or Next File from Panel or Find Window

When you have more multimedia files in one directory in panel or found files list in the Find window, you can use the View command for the first of them and then view all others in the same viewer window one by one using following commands:

  • Previous (the Backspace key): open the previous image file,
  • Next (the Space key): open the next image file,
  • First (the Shift+Backspace key): open the first image file,
  • Last (the Shift+Space key): open the last image file.

Note: all these commands work only until you change the content of panel or Find window from which you opened the viewer.

You can find all described commands in viewer submenu: File/Other Files.
