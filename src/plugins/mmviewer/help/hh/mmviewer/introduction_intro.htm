Open Salamander: Plugins: Multimedia Viewer: Introduction

Getting Started with Multimedia Viewer Plugin

With Multimedia Viewer you can view information about multimedia files. Parsers are licensed from the powerful tool for media cataloguing: Disk Explorer Professional. It works with the following multimedia file formats:

  • MP3, MP2 - MPEG 1 or MPEG 2 audio files, layer I, II, and III
    Displays MP3 ID3 Tag version 1 and 2
  • Ogg Vorbis, WMA, WAV
  • 669, IT, MOD, MTM, S3M, STM, XM - Tracker Modules

The modular architecture ensures that plugin can support new data formats as they become available.

See File Viewer and Menu Extension sections in Using Plugins for a description of basic work with plugin viewer and menu extension.
