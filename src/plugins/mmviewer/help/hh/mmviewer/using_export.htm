Exporting Information into HTML File

You can export information from multimedia files into HTML file using this command.

To export information from multimedia files into HTML file:

  1. Select multimedia files from which you want to export information.
  2. Choose Export Information into HTML File command:
    Menu:Plugins/Multimedia Viewer/Export Info into HTML...
  3. Specify name of exported HTML file.
  4. Choose the Save button.
