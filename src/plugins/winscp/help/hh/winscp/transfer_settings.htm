Transfer Settings

For several features of WinSCP you may want to configure transfer settings. This includes:

  • Uploading or downloading files.
  • Synchronizing files.
  • Function Keep remote directory up to date.

Generally you have two options how to configure the transfer settings. You can:

  • configure default transfer settings.
  • set ad hoc settings for particular operation (on Upload/Download, Synchronize and Keep remote directory up to date dialogs).
