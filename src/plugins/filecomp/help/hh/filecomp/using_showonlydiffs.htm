Showing Only Differences

You may want to see only the parts of files where are differences. You can specify how many equal lines should be displayed before and after the difference (as the context of the difference), see Number of context lines to show if only differences are viewed on the General tab in the Configuration dialog.

To show only differences:

File Comparator menu:View/Show Only Differences
File Comparator shortcut key:Ctrl+D
