Showing Detailed Differences

The plugin can also show detailed differences. It means that it highlights not only lines with differences, but directly added, changed, and deleted characters. If you want this to be always turned on, check Show detailed differences by default box on the General tab in the Configuration dialog.

To show detailed differences:

File Comparator menu:View/Detailed Differences
File Comparator shortcut key:F4
