Browsing Differences

The File Comparator usually finds many differences between compared files. To browse these differences you can use following commands:

  • First Difference (the Alt+Home key): go to the first difference,
  • Previous Difference (the Alt+Up Arrow key): go to the previous difference,
  • Next Difference (the Alt+Down Arrow key): go to the next difference,
  • Last Difference (the Alt+End key): go to the last difference.

You can find all described commands in the File Comparator window in the View menu.
