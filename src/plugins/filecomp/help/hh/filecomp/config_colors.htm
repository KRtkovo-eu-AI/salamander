Colors

Use this tab to set colors in the File Comparator window.

Choose the group of colors which you want to display/change: common colors, left panel colors, or right panel colors. Each color is displayed in its own row. Use the buttons in the row to change the color of text and background.

You can see all chosen colors in the preview at the bottom of this tab.
