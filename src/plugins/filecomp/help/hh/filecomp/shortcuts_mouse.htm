Mouse Shortcuts

There are many mouse shortcuts defined in File Comparator.

You can:
  • Select text by dragging mouse pointer over text
  • Select one word by double clicking on it with left mouse button
  • Select full lines by dragging mouse pointer over text and holding ctrl key down
  • Select one line by double clicking in it with left mouse button while holding ctrl key down
  • Focus difference by clicking on it with left mouse button
  • Focus and select whole difference by double clicking on it in line numbers column with left mouse button
  • Display context menu by clicking in text window with right mouse button
