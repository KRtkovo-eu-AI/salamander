Open Salamander: Plugins: Automation: Dialog Boxes

Automation Configuration

You can customize Automation plugin in this dialog. See Configuring Plugins for general information about configuration.

Options

Script Directories
Lists all the directories that should be searched for the scripts. Scripts found in these directories are added to the Automation Plugins menu and to the Script menu. Scripts placed in subdirectories are presented in submenus.
Enable script debugging
When checked, the Just-in-Time debugger installed on the system (e.g. Microsoft Script Debugger or Microsoft Visual Studio) can be attached to the script if it raises exception during the execution.
