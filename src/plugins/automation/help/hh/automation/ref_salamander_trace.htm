TraceE, TraceI Methods

These methods send a message to the Trace Server. The TraceE method sends the error level message, while the TraceI method sends the information level message.

Syntax

TraceE(message)
TraceI(message)

Parameters

message

Required. String to be sent to the Trace Server.

Return Values

These methods do not return a value.

Remarks

The Trace Server is the debugging tool available as a part of Preview Build Open Salamander installations or as a part of the Open Salamander Plugin SDK. In release versions of the Open Salamander these methods do nothing.
