WaitWindow Object

The WaitWindow object represents the wait window. Wait windows are used to inform users about lengthy operations when progress of the operation cannot be determined. The ProgressDialog object provides better alternative to inform users about progress of a lengthy operation.

Methods

Name Description
Hide Hides the wait window.
Show Shows the wait window.

Properties

Name Description
CanCancel Determines whether the user can cancel the progress dialog.
Delay Gets or sets initial delay before the wait window is displayed.
IsCancelled Determines whether the user has cancelled the dialog.
Text Sets or gets the text displayed inside the wait window.
Title Sets or gets the wait window title.
