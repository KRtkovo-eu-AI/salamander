Item Object

The Item object represents a panel item.

Properties

Name Description
Attributes Gets file attributes of the item.
DateLastModified Gets the date and time that the specified file was last modified.
Name Gets item file name.
Path Gets item full path.
Size Gets size of the item, in bytes.
