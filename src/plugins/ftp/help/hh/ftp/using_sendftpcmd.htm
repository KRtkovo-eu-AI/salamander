Sending FTP Command

You can send any FTP command directly to the FTP server opened in panel. Use this command to send auxiliary FTP commands.

To send FTP command to the FTP server opened in panel:

  1. Open the Send FTP Command dialog box:
    Menu:Plugins/FTP Client/Send FTP Command
    Context menu:Right-click in panel and select Send FTP Command.
  2. Specify options for sending, see options in the Send FTP Command dialog box.
  3. Enter the command you want to send.
  4. Click the OK button.
