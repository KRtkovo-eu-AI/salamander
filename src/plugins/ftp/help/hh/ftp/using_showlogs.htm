Showing FTP Logs

The communication between the FTP client and FTP server is stored in FTP logs. These logs can help you to understand and solve common problems with FTP connections. See FTP Logs window for details on logs.

To show FTP Logs window:

Menu:Plugins/FTP Client/Show Logs

To show FTP Logs window and focus log for FTP connection in panel:

Context menu:Right-click in panel and select Show Log.
