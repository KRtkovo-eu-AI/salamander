Connecting to FTP Server

This command allows you to connect to the FTP server. You can also store your favourite FTP servers to FTP bookmarks and use these bookmarks to quickly access these servers.

To connect to a bookmarked FTP server:

  1. Open the Connect to FTP Server dialog box:
    Menu:Plugins/FTP Client/Connect to FTP Server
    Shortcut key:Ctrl+Shift+F
    Change Drive menu:FTP Client
  2. Choose a bookmark to the FTP server you want to connect to.
  3. Click the Connect button.

To add bookmark to FTP server and connect to it:

  1. Open the Connect to FTP Server dialog box:
    Menu:Plugins/FTP Client/Connect to FTP Server
    Shortcut key:Ctrl+Shift+F
    Change Drive menu:FTP Client
  2. Click the New button to add new bookmark.
  3. Specify the name of new bookmark in the New Bookmark dialog box.
  4. Enter connection parameters (e.g. server address, initial path, and user name). See Connect to FTP Server dialog box for details on these parameters.
  5. Click the Connect button.

To connect to FTP server without using a bookmark:

  1. Open the Connect to FTP Server dialog box:
    Menu:Plugins/FTP Client/Connect to FTP Server
    Shortcut key:Ctrl+Shift+F
    Change Drive menu:FTP Client
  2. Focus the Quick Connect item (it is always the first item).
  3. Enter connection parameters (e.g. server address, initial path, and user name). See Connect to FTP Server dialog box for details on these parameters.
  4. Click the Connect button.
