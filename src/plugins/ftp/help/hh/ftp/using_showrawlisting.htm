Showing Raw Listing

When the FTP client gets the list of files and directories in the current directory on the FTP server, it gets only the plain text (raw) listing designed only for reading by a human (not for automatic parsing) from the FTP server. So it is possible that you encounter some extra format of this plain text listing which cannot be parsed automatically by the FTP client (or is not parsed properly). In such cases, you can display the plain text listing to see the content of the current directory as it is presented by the FTP server. You can also save this listing to disk and send it to Altap (email: support@altap.cz), we will try to support such listing.

To show raw listing of the current directory on the FTP server in panel:

Menu:Plugins/FTP Client/Show Raw Listing
Context menu:Right-click in panel and select Show Raw Listing.
