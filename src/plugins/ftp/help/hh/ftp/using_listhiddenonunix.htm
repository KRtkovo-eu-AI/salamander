Listing Hidden Files and Directories (Unix)

Some Unix FTP servers require an option "-a" behind the LIST command to list also hidden files and directories. You can easily turn on/off using of this option for the connection opened in panel.

To turn on/off listing of hidden files and directories (Unix) in the FTP connection in panel:

Menu:Plugins/FTP Client/List Hidden Files and Directories (Unix)
Context menu:Right-click in panel and select List Hidden Files and Directories (Unix).
