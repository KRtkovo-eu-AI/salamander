Changing Access Rights (Unix)

Use this command to change access rights (attributes) on Unix FTP servers.

To change access rights (attributes) on Unix FTP server:

  1. Select the files and directories whose access rights (attributes) you want to set.
  2. Open the Change Attributes dialog box:
    Menu:Files/Change Attributes...
    Shortcut key:Ctrl+F2
    Toolbar:Choose the Change Attributes button on the Top, Middle, or Bottom toolbar.
  3. Set new access rights (attributes) in the Owner, Group, and Others group boxes or the Numeric value edit box, see Change Attributes dialog box for details.
  4. Select the Include subdirectories check box if you want to process also files and directories in selected subdirectories.
  5. Select the Change attributes of files check box if you want to process files (otherwise it will skip all files).
  6. Select the Change attributes of directories check box if you want to process directories (otherwise it will skip all directories).
  7. Click the OK button.
