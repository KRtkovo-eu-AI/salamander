New/Edit Column

Use this dialog to add new or edit existing column in the server type. Use the New or Edit button in the Edit Server Type dialog box to open this dialog. See Getting Lists of Files and Directories from FTP Servers for details on columns.

Options

Identifier
Specify the column identifier.
Name
Specify the name of the column. This name will be displayed in panel. You can choose from the predefined names (advantage: it will be automatically translated to all language versions of the plugin) or enter some custom name (but it will not be possible to translate it).
Type
Choose the type of the column.
Empty value
Specify the empty value for the column. The format of this value varies depending on the column type and is displayed behind this edit box.
Description
Specify the description of the column. This description can be displayed in panel. You can choose from the predefined descriptions (advantage: it will be automatically translated to all language versions of the plugin) or enter some custom description (but it will not be possible to translate it).
Alignment
Choose the alignment of values displayed in the column.
