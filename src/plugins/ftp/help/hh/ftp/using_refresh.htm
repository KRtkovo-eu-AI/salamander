Refreshing Panel

Use this command to refresh the listing in panel. You can also use the standard command for refreshing, see Refreshing Panel. See the Use cache for viewed files and lists of files and directories on server option in Configuration/Defaults for details on reseting cache by the Refresh command.

To refresh listing of the current directory on the FTP server in panel:

Menu:Plugins/FTP Client/Refresh
Context menu:Right-click in panel and select Refresh.
