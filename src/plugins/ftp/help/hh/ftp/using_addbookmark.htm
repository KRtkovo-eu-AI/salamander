Adding Bookmark

You can add a bookmark for the current directory on the FTP server opened in panel. Later this bookmark can be used to quickly access the FTP server. See Connecting to FTP Server for details. Note: you can also add a bookmark without connecting to the FTP server, see Connecting to FTP Server and Organizing Bookmarks.

To add a bookmark:

  1. Activate the panel with opened connection to the FTP server.
  2. Choose the Add Bookmark command:
    Menu:Plugins/FTP Client/Add Bookmark
    Context menu:Right-click in panel and select Add Bookmark.
  3. Specify the name of new bookmark in the Add Bookmark dialog box.
  4. Place new bookmark in the Organize Bookmarks dialog box.
