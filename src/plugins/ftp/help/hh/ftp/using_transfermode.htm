Setting Transfer Mode

All starting copy and move operations between local machine and the FTP server (in both directions) are dependent on currently chosen file transfer mode. We have three transfer modes: Automatic, ASCII, and Binary. See the Default file transfer mode section in Configuration/Defaults for description of these modes. Note: you can preset the file transfer mode for an FTP bookmark in the Advanced Options dialog box, see the File transfer mode option.

To set transfer mode for FTP connection in panel:

Menu:Choose in the Plugins/FTP Client/Transfer Mode submenu.
Context menu:Right-click in panel and choose in the Transfer Mode submenu.
