Open Salamander: Plugins: FTP Client: Dialog Boxes

Move

Use this dialog box to move selected files and directories from the source panel into destination directory.

Options

Move to
Contains a path to the destination directory. It can be only full or relative disk path (moving to archives and plugin file systems are not supported). This area is initialized with the path to the target panel, but you can change it to any other directory. You can use the operation mask to rename processed filenames. Use the drop-down list to select previously used destination directory.
