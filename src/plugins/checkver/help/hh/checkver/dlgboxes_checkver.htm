Open Salamander: Plugins: Check Version: Dialog Boxes

Check For New Versions

Use this window to check for newer versions of Open Salamander and plugins. It connects to the ALTAP server on the Internet and gets information about current versions.

Options

Log window
It contains progress of checking for newer versions (after clicking on Check Now button). When all neccessary information are obtained, it shows the list of newer versions. You can click on module name to download its updated version. Or right click on module name to open context menu with all actions with module: download updated version, copy download URL to clipboard, and add this version or all versions of module to Filters (see Filters in the Configuration of Check Version dialog box). Click "show details" or "hide details" labels to show/hide new version details.
Configure
Use this button to open the Configuration of Check Version dialog box.
Check Now
It starts checking for newer versions.
