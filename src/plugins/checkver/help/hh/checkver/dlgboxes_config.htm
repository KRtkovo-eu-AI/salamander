Open Salamander: Plugins: Check Version: Dialog Boxes

Configuration of Check Version

You can customize behaviour of the Check Version plugin in this dialog. See Configuring Plugins for general information about configuration.

Options

Internet Connection Settings
Specify the type of your connection to the Internet and the protocol which is used to obtain data from the ALTAP server. Click on the Change button to see all types in the CheckVer Internet Connection Settings dialog box.
Open Check for New Versions window on Open Salamander start
Choose the period for automatic checking for newer versions (never, every day, every week, every month, every three months, or every six months).
When Check for New Versions window is opened, hide it and perform auto-connect
When checked, the plugin automatically connects to the ALTAP server and gets information about newer versions. Otherwise you must click on the Check Now button in the Check For New Versions window to start checking.
Close window if no new version is found, otherwise show window
When checked, the plugin is quiet until it finds some newer versions. If some new versions are available, it shows the Check For New Versions window with found updates. When not checked, you must close the Check For New Versions window manually every time using the Close button.
Filters
The Filters listbox contains versions of modules which are not interesting for you and should be filtered (ignored). You can add all versions of module (if no version is specified) or just one version of module (other versions will not be filtered). To add one or all versions of module to this list, right click the module name in the Log window in the Check For New Versions window when it displays the list of newer versions (results of checking).
Remove Selected Item From Filters
Click this button to remove selected item (one or all versions of module) from Filters listbox. Such item will not be filtered any longer.
Check for release versions of Open Salamander and plugins
When checked, you are interested in new release versions. Otherwise new release versions are ignored.
Check for beta versions of Open Salamander and plugins
When checked, you are interested in new beta versions. Otherwise new beta versions are ignored.
Check for preview build versions of Open Salamander and plugins
When checked, you are interested in new preview build versions. Otherwise new preview build versions are ignored.
Set defaults
Choose this button to set defaults in this dialog (it depends on selected type of the connection to the Internet).
