Checking For New Versions

Use the Check Version plugin to check for newer versions of Open Salamander and plugins. It connects to the ALTAP server on the Internet and gets information about current versions.

To check for new versions:

  1. Open the Check For New Versions window:
    Menu:Plugins/Check Version/Check For New Versions...
  2. Click the Check Now button.
