Open Salamander: Plugins: Check Version: Introduction

Getting Started with Check Version Plugin

Check Version allows you to easily check for newer versions of Open Salamander and plugins. It connects to the ALTAP server on the Internet and gets information about current versions. It can check for new versions automatically (with given period) and on the background (it shows window only if some newer version is available).

See Menu Extension section in Using Plugins for a description of basic work with menu extension.
