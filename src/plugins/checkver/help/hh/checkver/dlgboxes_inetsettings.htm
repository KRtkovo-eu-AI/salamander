CheckVer Internet Connection Settings

Use this dialog to specify the type of your connection to the Internet and the protocol which is used to obtain data from the ALTAP server.

Options

How do you connect to the Internet
Specify your type of connection to the Internet. Use the option local area network (LAN) for all permanent types of connections (ADSL, wifi, etc.).
Internet protocol
Choose the internet protocol used to download information from the ALTAP server. HTTP is almost always available. If not, use FTP or FTP in passive transfer mode. We recommend to use HTTP protocol because it allows to send version number of your Altap Salamander to our server and so it is the only way how to get information about updated or new plugins.
