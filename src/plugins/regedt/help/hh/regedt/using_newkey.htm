Open Salamander: Plugins: Registry Editor: Using Registry Editor Plugin

Creating New Key

When you need to create a new key in the specified key in the Windows registry, use this command.

To create a new key in registry:

  1. Open the key where you want to create a new key in panel.
  2. Open the Create Key dialog box:
    Menu:Plugins/Registry Editor/New Key...
    Menu:Commands/Create Directory...
    Shortcut key:F7
    Context menu:Open context menu in panel (right-click or Shift+Alt+F10) and choose New Key...
  3. Enter a key name.
  4. Choose the OK button.
