Open Salamander: Plugins: Registry Editor: Using Registry Editor Plugin

Exporting Registry Key

You can export a key from the Windows registry to a disk file.

To export a focused key from registry to a disk file:

  1. Focus the key you want to export in panel.
  2. Open the Export dialog box:
    Menu:Plugins/Registry Editor/Export...
    Context menu:Open context menu for the focused key in panel (right-click or Shift+F10) and choose Export...
  3. Enter a resulting filename.
  4. Choose the OK button.

To export a registry key opened in panel to a disk file:

  1. Open the key you want to export in panel.
  2. Open the Export dialog box:
    Context menu:Open context menu in panel (right-click or Shift+Alt+F10) and choose Export...
  3. Enter a resulting filename.
  4. Choose the OK button.
