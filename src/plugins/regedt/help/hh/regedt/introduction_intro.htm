Open Salamander: Plugins: Registry Editor: Introduction

Getting Started with Registry Editor Plugin

With the Registry Editor plugin you can browse the Windows registry in panel; rename, copy, move, and delete keys and values; view and edit values; search in the registry (also using advanced criteria and regular expressions); and export keys.

See File System and Menu Extension sections in Using Plugins for a description of basic work with plugin file systems and menu extensions.
