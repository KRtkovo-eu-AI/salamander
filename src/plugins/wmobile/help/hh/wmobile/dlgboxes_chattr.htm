Change Attributes

Use this dialog box to change attributes of selected files and directories.

Dialog Box Options

Set Attributes
Each attribute is represented by one check box with the following meaning:
Attribute is set for all selected files and directories.
Attribute is not set for any of the selected files and directories.
Attribute is set for some, but not all, of the selected files and directories.
To set attribute, select its check box. To clear attribute, clear its check box. If you want to leave attribute as is, set its check box to the third state.
Change Date and Time
Allows you to specify last modification, creation, and last access times and dates of selected files and directories. Use the Set Current button to set the current time and date to ticked boxes with time and date.
Include subdirectories
When checked, operation includes files and directories contained within selected directories.
