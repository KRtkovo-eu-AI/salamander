Selecting Scanner Source

If you have more scanners attached to your computer, use this command to choose the scanner you want to use in the Scan Image command. This command is only in 32-bit (x86) version, see Scan Image command for more details.

To select scanner source:

  1. Open Select Source dialog box:
    Menu:Plugins/PictView/Select Scanner Source...
    Viewer menu:File/Select Scanner Source...
  2. Choose your scanner from the list.
  3. Choose Select button.
