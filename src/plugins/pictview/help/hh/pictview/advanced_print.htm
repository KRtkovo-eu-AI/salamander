Printing Image

Use this command for sending the image opened in the viewer to a printer. You can choose position and size of the image.

To print image:

  1. Open the Print dialog box:
    Viewer menu:File/Print...
    Viewer shortcut key:Ctrl+P
  2. Use the Setup button to select printer and paper format and orientation.
  3. If you want to print only the selected area of the image (see how to select the area in Using Mouse in Viewer), check the Print selected area check box.
  4. Enter position and size of image.
  5. Choose the OK button.
