Focusing File in Panel

You can quickly switch from the viewer to the active panel of Open Salamander and focus the currently opened image file. It's handy for applying any other command to this image file directly in the panel.

To focus the image file in the panel:

Viewer menu:File/Focus
Viewer shortcut key:F
Viewer context menu:Right-click image and choose Focus
