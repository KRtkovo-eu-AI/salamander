Tools

Customization of behaviour of tools.

Select tool

Area selected when holding the Shift key
When using the Select tool, a rectangle with predefined aspect ratio gets selected if the Shift key is held down. By default, a square is selected. However, here you can specify another rectangle as width : height. Several often used sizes of computer & TV screens and photographs are predefined, plus you can choose any custom aspect ratio.

TIP: Only integral numbers can be used for Other. To bypass this limitation, multiply your numbers by 10 or 100. For example, if you need a very specific ratio 1:1.234, specify 1000:1234 instead.

Other options

Pipette picks colors in hexadecimal
See Pick Color mode in Using Mouse in Viewer. When checked, the pipette tool shows colors as hexadecimal numbers (useful e.g. when editing HTML pages).
