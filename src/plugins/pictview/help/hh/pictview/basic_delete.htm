Deleting File

You can easily delete the currently opened image file from the disk. More precisely you can send this file to Recycle Bin or when you press (and hold) the Shift key, you can delete it permanently.

To delete file:

Viewer menu:File/Delete
Viewer shortcut key:Delete or Shift+Delete
Viewer context menu:Right-click image and choose Delete
