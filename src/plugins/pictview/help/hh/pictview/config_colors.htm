Colors

Use this tab to set colors in the viewer.

Options

Window mode: Background
In window mode: set here the color of the part of view which is not used by the image.
Window mode: Transparent
In window mode: set here the color of the transparent parts of the image.
Full screen mode: Background
In the full screen mode: set here the color of the part of the screen which is not used by the image.
Full screen mode: Transparent
In the full screen mode: set here the color of the transparent parts of the image.
