Renaming File

You can easily rename the currently opened image file on the disk.

To rename file:

  1. Open the Rename File dialog box:
    Viewer menu:File/Rename...
    Viewer shortcut key:F2
    Viewer context menu:Right-click image and choose Rename...
  2. Enter a new filename.
  3. Click the OK button.
