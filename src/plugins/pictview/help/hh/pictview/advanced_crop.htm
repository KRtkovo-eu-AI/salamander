Cropping Image

Use this command to replace the image in the viewer by the selected part of this image.

To crop image:

  1. Select a part of the image you want to keep. See how to do it in Using Mouse in Viewer.
  2. Crop image:
    Viewer menu:Edit/Crop
    Viewer shortcut key:C
