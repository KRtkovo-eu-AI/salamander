Open Salamander: Plugins: PictView: Dialog Boxes

Copy File To

Use this dialog to copy the image file currently opened in the viewer to one of five target directories.

Options

1, 2, 3, 4, and 5
You can specify up to five target directories in these editboxes. But only the current one will be used. The current target directory is the only one which is enabled (others are grayed).
Browse
Displays the Browse dialog box so you can select the target directory on the disk.
