Scanning Image

You can scan documents directly to the viewer.

32-bit (x86) version:

If you have more scanners attached to your computer, use the Select Scanner Source command to choose the appropriate scanner. 32-bit (x86) version uses TWAIN interface to scanners. TWAIN x86 provides also WIA interface (Windows Image Acquisition), so you should easily use any scanner.

64-bit (x64) version:

If you have more scanners attached to your computer, you will choose the appropriate scanner whenever you start scanning. 64-bit (x64) version uses WIA interface (Windows Image Acquisition). TWAIN x64 is not directly supported by Windows and also supported only by some scanner manufacturers. If you cannot connect to your scanner from 64-bit (x64) version try to connect it from 32-bit (x86) version, it will probably work well with your scanner.

To scan image:

  1. Start scanning of image:
    Menu:Plugins/PictView/Scan Image...
    Viewer menu:File/Scan Image...
    Viewer shortcut key:W
  2. Scanned image will be displayed in PictView viewer window. You can use Save As command from menu File to save this picture to file.
