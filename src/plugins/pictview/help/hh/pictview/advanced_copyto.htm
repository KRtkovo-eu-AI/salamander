Copying File

This command allows you to copy the image file currently opened in the viewer to one of five target directories. You can specify the target directory each time you make the copy or you can quickly reuse the target directories used in previous copy commands.

To copy file:

  1. Open the Copy File To dialog box:
    Viewer menu:File/Copy To...
    Viewer shortcut key:X
  2. Select the target directory.
  3. Choose the OK button.
