Keyboard & Mouse

Customization of keyboard and mouse use.

Options

Page Up & Page Down keys, mouse wheel
Specifies what actions should be taken when you use the Page Up or Page Down key or mouse wheel. Select one of the following options:
Option Meaning
Used for image scrolling Use the Page Up and Page Down keys and mouse wheel to scroll the view.
Used to access previous/next image Use the Page Up and Page Down keys and mouse wheel to open previous or next image file from the panel or the Find window. See Opening Previous or Next File from Panel or Find Window for details.
