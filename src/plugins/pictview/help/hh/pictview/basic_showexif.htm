Showing EXIF Information

Most digital cameras use the EXIF format (Exchangeable Image File Format) to store information about the image and camera (camera orientation, ISO rating, focal length, exposure time, etc.) directly in the JPEG, TIFF, or RAW image file. You can display EXIF information for the image opened in the viewer.

To show EXIF information:

Viewer menu:File/Show EXIF...
Viewer shortcut key:E
Viewer context menu:Right-click image and choose Show EXIF...
