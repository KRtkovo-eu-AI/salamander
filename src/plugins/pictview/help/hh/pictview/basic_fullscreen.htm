Viewing in Full Screen

If you want to fully enjoy viewing of the image, try to view it in the full screen. In this view you can see only the image, everything else is hidden (including mouse cursor, it disappears if mouse is without movement for couple of seconds).

To view the image in the full screen:

Viewer menu:View/Full Screen
Viewer shortcut key:F11
Viewer context menu:Right-click image and choose Full Screen

To exit the full screen view:

Viewer shortcut key:F11
Viewer context menu:Right-click image and choose Full Screen
