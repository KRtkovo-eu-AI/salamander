Selecting Whole Image

You can set the selected area of the image to the entire image. You can also cancel selection.

To select whole image:

Viewer menu:Edit/Select All
Viewer shortcut key:Ctrl+A

To cancel selection:

Viewer menu:Edit/Deselect All
Viewer shortcut key:Ctrl+D
