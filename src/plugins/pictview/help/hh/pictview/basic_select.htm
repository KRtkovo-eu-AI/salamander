Selecting File in Panel or Find Window

You can open an image file in the viewer from a panel or found files list in Find window using the View command. Then you can select (if not selected) or unselect (if selected) the image file in the panel or found files list in Find window directly from the viewer.

Note: this command works only until you change the content of the panel or Find window from which you opened the viewer.

To select/unselect the image file in panel or Find window:

Viewer menu:File/Select/Unselect Opened File in Panel or Find
Viewer shortcut key:Insert
Viewer context menu:Right-click image and choose Select/Unselect Opened File in Panel or Find
