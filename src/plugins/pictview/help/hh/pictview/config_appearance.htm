Open Salamander: Plugins: PictView: Dialog Boxes: Configuration

Appearance

Allows you to change the appearance of the PictView viewer. See Configuring Plugins for general information about configuration.

Options

Window size
Specifies what size should be the viewer window when you open it. Select one of the following settings:
Option Meaning
Same as of Open Salamander Viewer window should be the same size as the main window of Open Salamander.
Larger if needed but no smaller Viewer window should be the same size or bigger than the main window of Open Salamander. It's bigger if the image in 100% zoom (1:1) needs bigger window.
As needed Viewer window should be the smallest possible to view the image in 100% zoom (1:1).
Default zoom
Specifies what zoom should be used in the viewer window when you open it. Select one of the following settings:
Option Meaning
Fit whole (no scrollbars) It calculates the zoom percentage to show the whole image in the viewer window.
Fit to width (no horizontal scrollbar) It calculates the zoom percentage to show the whole width of the image in the viewer window (it avoids horizontal scrolling, but vertical scrolling can be still needed).
Original size Use 100% zoom percentage (one pixel of the image is one pixel in the viewer window).
Full screen View image in full screen view mode. It calculates the zoom percentage to show the whole image in the full screen view.
Show image file path in titlebar
When checked, the viewer titlebar contains full path to image file. Otherwise it contains only name of the image file (without full path).
