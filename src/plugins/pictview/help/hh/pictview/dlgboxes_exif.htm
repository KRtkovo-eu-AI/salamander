EXIF

Use this dialog to view EXIF information for the image opened in the viewer.

Options

EXIF tags
The list of EXIF tags with values gained from the image file.
EXIF tag description
The description of the focused tag in the list of EXIF tags.
Group highlighted items
When checked, the highlighted items (tags) in the list of EXIF tags are grouped at the beginning of the list.
Highlight
When the button is checked (down), the focused tag in the list of EXIF tags is highlighted. Click this button to change highlight of this tag.
Copy
Use this button to copy the value of focused tag in the list of EXIF tags to the clipboard.
