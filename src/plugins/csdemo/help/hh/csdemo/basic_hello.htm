Managed Hello Command

The Show C# Demo Window command demonstrates how a managed WinForms window can be opened from a native plugin. The dialog is implemented in the CSDemo.Managed.dll assembly and is hosted by the C++ bridge.

To open the managed window:

Menu:Plugins/C# Demo/Show C# Demo Window
Shortcut key:Ctrl+Shift+M

If the window fails to appear, verify that the managed assembly is next to csdemo.spl and that the .NET Framework 4.x runtime is available.
