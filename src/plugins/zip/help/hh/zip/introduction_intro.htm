Open Salamander: Plugins: ZIP: Introduction

Getting Started with ZIP Plugin

ZIP plugin adds a support for ZIP archives. You can pack files and directories to ZIP archives; browse, test and unpack ZIP archives; delete files and directories from ZIP archives; create self-extracting ZIP archives (in several language versions); create multi-volume (spanned) ZIP archives (also self-extracting; and even on fixed disks); and view/edit ZIP archive comment. Files in ZIP archives (also self-extracting) can be encrypted.

See Archiver and Menu Extension sections in Using Plugins for a description of basic work with plugin archiver and menu extension.
