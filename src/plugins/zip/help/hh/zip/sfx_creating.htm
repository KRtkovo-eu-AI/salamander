Creating Self-Extracting Archive

There are tree ways how to create self-extracting archive:

From ZIP plugin menu

If you have already created a ZIP archive, you can convert it to self-extracting archive by command Create Self-Extracting Archive from menu Plugins/ZIP.

Specifying an option in Create New Archive dialog

When you pack files to a new archive (using Pack command from menu Files), ZIP plugin displays Create New Archive dialog. To create self-extracting archive check Create self-extracting archive in this dialog. If this dialog doesn't appear, check Display dialog with extended options before packing in your ZIP Configuration.

Creating self-extracting archive from command prompt

This feature is not available in ZIP plugin itself but is provided by a standalone application Zip2Sfx which is placed in plugins\zip\zip2sfx subdirectory of Open Salamander's installation directory.
