Self-Extractor Command Line Parameters

User can start self-extracting archive with command line option -s. With this option self-extracting archive starts in safe mode. It means that following self-extractor options are forced: Allow user to change the directory is checked, Start extraction automatically is cleared and Hide main self-extractor's dialog is cleared. So it displays the main self-extractor dialog box and lets user to specify Extract to directory.
