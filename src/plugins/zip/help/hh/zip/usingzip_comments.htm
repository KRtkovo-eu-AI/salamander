Open Salamander: Plugins: ZIP: Using ZIP Plugin

Archive Comment

ZIP archive can contain two types of comments. Comments to individual compressed files and comment to whole archive. ZIP plugin enables you to view, add and edit only archive comment.

To view/edit archive comment:

  1. Select the ZIP archive files for which you want to view/edit comment.
  2. Choose Archive Comment command from menu Plugins/ZIP.
