Showing DiskMap

Use this command to open DiskMap for current path in panel. To display DiskMap for whole disk you have to be in the root directory.

To show DiskMap:

Menu:Plugins/DiskMap/Show DiskMap
Shortcut key:Ctrl+Shift+D
