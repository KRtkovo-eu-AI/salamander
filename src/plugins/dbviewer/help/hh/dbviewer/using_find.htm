Searching

You can search in displayed values in the viewer. See the following commands:

  • Find (the Ctrl+F key): opens the Find dialog box where you can specify the text you want to search for and search options,
  • Find Next (the Ctrl+N key): repeat searching,
  • Find Previous (the Ctrl+P key): repeat searching but in the opposite direction.

You can find all described commands in viewer menu Edit.
