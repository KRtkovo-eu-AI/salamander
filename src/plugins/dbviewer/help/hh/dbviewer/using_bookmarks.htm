Using Bookmarks

You can bookmark any item in the viewer. It allows you to find this item later very quickly. Bookmarked items have red marks in upper right corners. Use the following commands to work with bookmarks:

  • Toggle Bookmark (the Ctrl+F2 key): bookmark focused item or remove bookmark from focused item,
  • Next Bookmark (the F2 key): go to the next bookmark,
  • Previous Bookmark (the Shift+F2 key): go to the previous bookmark,
  • Clear All Bookmarks (the Ctrl+Shift+F2 key): clear all bookmarks.

You can find all described commands in viewer menu Edit.
