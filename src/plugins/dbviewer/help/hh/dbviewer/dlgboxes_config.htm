Database Viewer Configuration

Use this dialog box to customize the viewer. See Configuring Plugins for general information about configuration.

Options

Font
Use this button to choose the font for displaying all values in the viewer.
Window position: Save on close
When checked, new viewer window will be opened on the same place and in the same size as the last closed viewer window.
Window position: Set to position of Salamander's main window
When checked, new viewer window will be opened on the same place and in the same size as the main window of Open Salamander.
