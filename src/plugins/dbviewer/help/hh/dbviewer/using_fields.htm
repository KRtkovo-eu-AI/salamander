Displaying Columns

Some columns in the viewer can be hidden, and you can also change the order of columns.

To show/hide or reorder columns:

  1. Open the Fields dialog box:
    Viewer menu:View/Fields...
    Viewer shortcut key:Ctrl+I
  2. Tick or clear boxes displayed before field names to show or hide appropriate column in the viewer.
  3. Use the Alt+Up and Down Arrow keys to change the order of fields (columns) in the viewer.
  4. Click the OK button.
