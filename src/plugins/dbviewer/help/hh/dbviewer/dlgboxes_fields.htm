Open Salamander: Plugins: Database Viewer: Dialog Boxes

Fields

Fields (columns) of the displayed file are listed in this dialog box. You can see data type, width and number of decimals (valid only for numbers) for each field. Tick or clear boxes displayed before field names to show or hide appropriate column in the viewer. Use the Alt+Up and Down Arrow keys to change the order of fields (columns) in the viewer.

Options

Restore Defaults
Use this button to show all fields (columns) in the viewer and to return to the original order of fields (columns) in the viewer.
