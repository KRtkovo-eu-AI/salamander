Setting CSV Options

Use this command to customize reading of CSV files. You can choose a character which separates values in rows, specify if text values are surrounded by some quotes or not, and select how to use a first row in CSV file (for column names or as data). See CSV Options dialog box for details.

To set the CSV options:

Viewer menu:Options/CSV Options...
