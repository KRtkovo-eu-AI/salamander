Getting Started with Database Viewer Plugin

With Database Viewer you can view basic database files. You can also search in them, copy one or group of items to the clipboard, and bookmark items for fast access.

Supported formats:

  • dBase II, dBase III, dBase IV, dBase V, Visual dBase 7, FoxPRO (*.DBF)
  • Comma separated values (*.CSV)
  • Tab delimited texts (*.TXT)

See File Viewer section in Using Plugins for a description of basic work with plugin viewer.
