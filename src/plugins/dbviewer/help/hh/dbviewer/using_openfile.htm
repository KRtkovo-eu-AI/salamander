Opening File

Use this command to open another database file in the viewer.

Note: it's usually simpler to close the current viewer window (the Escape key), find the database file in panel or in the Find window, and open it in the viewer (the F3 key).

To open file:

  1. Open the Open dialog box:
    Viewer menu:File/Open...
    Viewer shortcut key:Ctrl+O
  2. Select the file.
  3. Click the Open button.
