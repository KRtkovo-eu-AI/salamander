Copying Items to Clipboard

You can copy selected items to the clipboard. Items are copied as tab-delimited text, so you can paste them to Microsoft Excel, Microsoft Word, Notepad, etc.

To copy selected items to the clipboard:

  1. Select items you want to copy to the clipboard. See Selecting Items for details.
  2. Copy selected items to the clipboard:
    Viewer menu:Edit/Copy
    Viewer shortcut key:Ctrl+C
