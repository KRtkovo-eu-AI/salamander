Recovering Deleted Files and Directories

You can recover deleted files and directories from NTFS, FAT12, FAT16, FAT32, and exFAT partitions. See Introduction for details on the principles of recovering.

To recover deleted files and directories:

  1. Open the Undelete dialog box:
    Menu:Plugins/Undelete/Recover Deleted Files and Directories
    Shortcut key:Ctrl+Shift+U
  2. Choose a volume or a disk image you want to analyze.
  3. Choose the OK button.
  4. Select files and directories you want to recover.
  5. Use the Copy command (the F5 key) to recover them to the specified target directory (should be on another volume).
