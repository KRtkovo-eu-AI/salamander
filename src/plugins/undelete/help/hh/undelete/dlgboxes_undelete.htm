Open Salamander: Plugins: Undelete: Dialog Boxes

Undelete

Choose a volume or a disk image you want to analyze.

Options

Disk Image
Clear this check box if you want to analyze a volume. Choose the volume from the list of available volumes. Select this check box if you want to analyze a disk image. Enter file name of the disk image in the edit box below.
Scan vacant clusters on FAT partitions
When checked, the plugin scan also vacant clusters of FAT volumes when for looking for directories. It is ignored on NTFS and exFAT volumes.
Show existing files and directories
When checked, both deleted and existing files and directories will be displayed.
Show file with zero size
When checked, also files with zero size will be displayed.
Show empty directories
When checked, also empty directories will be displayed.
Show metafiles
When checked, there will be special {Metafiles} virtual folder, containing some file system metafiles. You can ignore this folder for ordinary recovery tasks.
