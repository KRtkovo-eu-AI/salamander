Open Salamander: Shortcuts

Context Menus

There are many context menus defined in Open Salamander. Context menus (also referred to as context or pop-up menus) include only commands that apply to the selected object (or objects) and its context. Most of windows and controls in Open Salamander can be controlled from context menu.

To open context menu, point the mouse cursor above desired object and right-click it. When the keyboard focus is on an item that can also be selected (such as an item in the panel or text in the Command line), you can press Shift+F10 or the Application key to display the context menu for the selected item or items.
