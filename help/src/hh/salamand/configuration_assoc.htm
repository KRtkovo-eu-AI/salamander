Open Salamander: Dialog Boxes: Configuration

Archives Associations in Panels

Allows you to associate extensions to the archivers.

Dialog Box Options

File Extensions
Displays a list of extensions associated with archivers. More extensions can be associated with one archiver. Extensions are separated by semi-colon (;), character '#' stands for any digit ('0' ... '9').
List and Unpack
Displays the archiver name associated to the selected extensions in File Extensions. This archiver will be used for listing and unpacking of archive.
Delete and Pack
Displays the archiver name associated to the selected extensions in File Extensions. This archiver will be used for deleting files from archive and packing files into archive.
