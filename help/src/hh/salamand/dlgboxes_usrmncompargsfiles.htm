Open Salamander: Dialog Boxes

Choose Files to Compare

Specify files to compare in this dialog.

Dialog Box Options

Compare file
Specify the first file to compare. You can browse for this file using the button behind this field.
With file
Specify the second file to compare. You can browse for this file using the button behind this field.
Do not display this dialog if you have both names to compare
When checked, this dialog is not displayed when both files were specified in panels. See Customizing User Menu for details. You can change this option later in Configuration/Confirmations
