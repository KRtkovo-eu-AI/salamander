Opening Directory You've Worked In Recently

List of working directories provides easy way to access directories you've worked in recently. To open recent directory open list of working directories and choose the directory you want to go to.

The history can be shared between both panels or maintained separately for each tab. Use the options on the History configuration page to decide whether the list follows you everywhere or stays local to the active tab when tabbed panels are enabled.

To open history list:

Menu:Commands/List of Working Directories
Shortcut key:Alt+F12
Mouse:In the Directory line press drop-down arrow between current directory and free space.
