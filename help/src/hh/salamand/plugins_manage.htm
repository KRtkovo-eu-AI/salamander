Open Salamander: Using Plugins

Managing Plugins

You can add, remove, and configure plugins in the Open Salamander. See Plugins Concepts for more information on plugins. To manage plugins use the Plugins Manager dialog box. All actions, such as adding or removing plugins, apply only to the Open Salamander configuration. No plugin file will be affected or removed from your disk. If you remove plugin, you can add it back again later.

To add a new plugin:

  1. Open the Plugins Manager dialog box: in the Plugins menu choose Plugins Manager.
  2. Choose the Add button.
  3. Find the plugin file (*.SPL) you want to Add and click the OK button.

To remove an installed plugin:

  1. Open the Plugins Manager dialog box: in the Plugins menu choose Plugins Manager.
  2. In the Installed Plugins list, select the plugin you want to remove.
  3. Choose the Remove button.

To configure an installed plugin:

  1. Open the Plugins Manager dialog box: in the Plugins menu choose Plugins Manager.
  2. In the Installed Plugins list, select the plugin you want to configure.
  3. Choose the Configure button.
