Open Salamander: Dialog Boxes: Password Manager

Change Master Password

Use this dialog box to set, change, or remove a Master Password for Open Salamander Password Manager.

Dialog Box Options

Current password
If a Master Password is already set, you will need to enter it to this field in order to change or remove the Master Password.
Enter new password
Enter new Master Password to set or change it. Leave this field clear to remove Master Password.
Re-enter password
Enter the new Master Password again to this field to confirm.
