Open Salamander: Dialog Boxes

Choose Directories to Compare

Specify directories to compare in this dialog.

Dialog Box Options

Compare directory
Specify the first directory to compare. You can browse for this directory using the button behind this field.
With directory
Specify the second directory to compare. You can browse for this directory using the button behind this field.
Do not display this dialog if you have both names to compare
When checked, this dialog is not displayed when both directories were specified in panels. See Customizing User Menu for details. You can change this option later in Configuration/Confirmations
