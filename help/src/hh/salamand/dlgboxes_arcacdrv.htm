Archivers Autoconfiguration Drives

Use this dialog box to specify which drives and directories will be searched for external archivers.

Dialog Box Options

Edit paths to search for external archivers
Displays a list of paths that will be searched for external archivers. You can add, delete, modify and change search order of the paths in this window. Both local and network paths are allowed (in UNC syntax).
