Open Salamander: Window Components

Window Menu

The Window menu contains the following commands:

  • Restore: when the window is maximized, this command restores it to its original size,
  • Move: allows to move window using only the Arrows keys,
  • Size: allows to resize window using only the Arrows keys,
  • Minimize (Shift+Escape): minimizes the window,
  • Maximize (the Ctrl+Shift+F11 key): maximizes the window,
  • Close: exits Open Salamander.

You can find this menu in the upper left corner of the main window (click on the application icon).
