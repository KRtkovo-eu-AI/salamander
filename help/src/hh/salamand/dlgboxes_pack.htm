Open Salamander: Dialog Boxes

Pack

Use this dialog box to create archive from selected files and directories.

Dialog Box Options

Pack to archive
Enter the location of archive file you want to pack selected files and directories to.
Packer
Specify packer which will be used. You can configure packers in Configuration/Archives/Packers in Pack Dialog.
Delete files from disk after packing to archive
When selected, files will be deleted after packing is successfully done.
