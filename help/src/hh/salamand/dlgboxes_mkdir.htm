Open Salamander: Dialog Boxes

Create Directory

Use this dialog box to create new directory.

Options

Directory Name
Type the name for the new directory. Name can be entered either as relative path to the current directory or as full path. You can create more directories at once (e.g. if you enter C:\a\b\c and C:\a does not exist, three directories will be created).
