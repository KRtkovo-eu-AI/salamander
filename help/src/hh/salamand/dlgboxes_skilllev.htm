Open Salamander: Dialog Boxes

User's Skill Level

Use this dialog to reduce number of items in menu according to your current skill level to simplify work with Open Salamander.

Dialog Box Options

Beginning
Set this level if you are starting with Open Salamander. Menu will contain only the basic commands.
Intermediate
Set this level if you think that you have already learned all basic commands. Menu will contain commands for intermediate users.
Advanced
Set this level if you want to use the full power of Open Salamander. Menu will contain all commands.
