Open Salamander: Dialog Boxes: Configuration

Confirmations

Specifies whether confirmation messages are displayed.

Dialog Box Options

File or directory delete
Clear this check box if you don't want to receive a confirmation message before deleting files.
Non-empty directory delete
Clear this check box if you don't want to receive a confirmation message before deleting non-empty directory. When you delete a directory, all the files and directories within it are deleted.
File overwrite
Clear this check box if you don't want to receive a confirmation message before replacing a file. It is recommended that you leave this option selected so that you don't inadvertently write over files you want to keep.
System or hidden file delete
Clear this check box if you don't want to receive a confirmation message before deleting system and hidden files.
System or hidden directory delete
Clear this check box if you don't want to receive a confirmation message before deleting system and hidden directories.
System or hidden file overwrite
Clear this check box if you don't want to receive a confirmation message before replacing system and hidden files.
Directories merge
Clear this check box if you don't want to receive a confirmation message before merging a directory (which you copy or move) with the existing directory (in the target directory).
NTFS compress or uncompress
Clear this check box if you don't want to receive a confirmation message before compressing or uncompressing files on NTFS drive.
NTFS encrypt or decrypt
Clear this check box if you don't want to receive a confirmation message before encrypting or decrypting files on NTFS drive.
Drag and Drop operations
When checked, all drag and drop operations will be confirmed using preceding context menu.
User Menu: always show dialog with names to compare
This applies to User Menu command which uses variables for files or directories compare (First File to Compare, Second File to Compare, etc.). When checked, the dialog with names to compare will be displayed even if both names were specified in panel. Otherwise, this dialog is displayed only if you have to specify at least one name for compare.
Compare Directories: ignored differences of exactly 1 or 2 hours
When checked and the Compare Directories command detects pairs of files with time differences of exactly one or two hours, and these differences should be ignored, it ignores them, but it also shows a note about how many pairs of files have such time differences. Otherwise, this note is not displayed. See Compare Directories: ignore time differences of exactly one or two hours check box in Configuration/General.
Compare Directories: found differences of exactly 1 or 2 hours
When checked and the Compare Directories command detects pairs of files with time differences of exactly one or two hours, and these differences should not be ignored, it informs you that you can change settings to ignore them (to bypass DST related file time shifts). Otherwise, this hint is not displayed. See Compare Directories: ignore time differences of exactly one or two hours check box in Configuration/General.
Archive is about to close... (only if editing files from archive)
Clear this check box if you don't want to receive a confirmation message before closing archive from which files were edited or opened (files may be changed).
Do you want to close Find?
Clear this check box if you don't want to receive a confirmation message before closing Find window (see Finding Files and Directories).
Do you want to stop searching?
Clear this check box if you don't want to receive a confirmation message before stopping of searching in Find window (see Finding Files and Directories).
Do you want to create target path?
Clear this check box if you don't want to receive a confirmation message before creating specified non-existing target path in Copy/Move operation.
Do you want to turn on Always on Top?
Clear this check box if you don't want to receive a confirmation message before turning on Always on Top (see Setting Window Always on Top).
Do you want to close Open Salamander?
Clear this check box if you don't want to receive a confirmation message before closing Open Salamander application.
Do you want to Email selected files?
Clear this check box if you don't want to receive a confirmation message before sending selected files by email (see Emailing Files).
Do you want to add files into existing archive?
Clear this check box if you don't want to receive a confirmation message before adding files to an existing archive. When checked, you can choose if to add files to existing archive or create a new archive (overwrite the existing archive).
The directory doesn't exist. Do you want to create it?
Clear this check box if you don't want to receive a confirmation message before creating of a directory in which some operation should be done (but which is not existing). This happens when creating a directory structure in Create Directory command (e.g. creating C:\a\b if c:\a does not exist)
