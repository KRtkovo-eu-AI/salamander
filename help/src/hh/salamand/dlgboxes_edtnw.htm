Open Salamander: Dialog Boxes

Edit New File

Use this dialog box to create new file and open it in associated editor.

Dialog Box Options

New File Name:
Type the name for the new file. If you want to create the file in other than current directory, you can type the full path of the new file.
Options (click the button behind the New File Name field)
It is possible to store current filename as default using the Save current file name as default command or return to original filename using the Revert default file name command. Default filename will be used when you open this dialog box next time.
