Open Salamander: Dialog Boxes: Configuration

Tabs

Fine-tune how panel tabs present their captions on this page. The page is visible only when tabbed panels are enabled.

Dialog Box Options

Display current path in tab name as
Choose how much of the active directory path should appear in each tab caption:
  • Directory Name Only – shows just the name of the current directory.
  • Shortened Path – uses an abbreviated path that preserves the drive and the most significant parent directories.
  • Full Path – displays the full directory path without shortening.
The selected mode applies to all tabs on both panels.
