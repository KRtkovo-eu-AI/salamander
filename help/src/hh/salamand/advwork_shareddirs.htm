Displaying Shared Directories

Shared directories are important when you need to share your files with others on the network. Users from other computers are able to connect to your shared directories (if they have proper access rights), see Connecting to and Disconnecting from a Network Drive for details.

To display shared directories:

  • Open the Shared Directories dialog box:
    Menu:Commands/Shared Directories...
    Shortcut key:Ctrl+Shift+F9
    Toolbar:Choose the Shared Directories button on the Top or Middle toolbar.
  • You can also stop sharing of directory and find shared directory from this dialog box.

Remarks

To start sharing use Sharing tab in Properties dialog of directory you want to share, see Opening Properties of Files and Directories for details.
