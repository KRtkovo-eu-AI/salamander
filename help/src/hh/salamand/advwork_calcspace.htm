Calculating Occupied Space

This command allows you to calculate space occupied by files and directories. You can also estimate the needed space on disk with the given cluster size.

To calculate the occupied space or estimate the needed space:

  1. Select the files and directories whose occupied space you want to calculate.
  2. Open the Calculate Occupied Space dialog box:
    Menu:Commands/Calculating Occupied Space
    Shortcut key:Alt+F10
    Toolbar:Choose the Calculating Occupied Space button on the Bottom toolbar.
  3. If you want to estimate the needed space, specify the cluster size.
