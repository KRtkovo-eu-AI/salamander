Open Salamander: Glossary

M

maximize

To enlarge a window to its maximum size by using the Maximize button (at the right of the title bar) or the Maximize command on the window menu. See also minimize.

Maximize button

The small button containing a window icon at the right of the title bar. Mouse users can click the Maximize button to enlarge a window to its maximum size. Keyboard users can use the Maximize command on the window menu.

menu

A list of available commands in an application window. Menu names appear in the menu bar near the top of the window. The window menu, represented by the program icon at the left end of the title bar, is common to all applications for Windows. To open a menu, click the menu name.

menu bar

The horizontal bar containing the names of all the application's menus. It appears below the title bar.

minimize

To reduce a window to a button on the taskbar by using the Minimize button (at the right of the title bar) or the Minimize command on the window menu. See also maximize.

Minimize button

The small button containing a short line at the right of the title bar. Mouse users can click the Minimize button to reduce a window to a button on the taskbar. Keyboard users can use the Minimize command on the window menu.
