Open Salamander: Dialog Boxes

Plugin Keyboard Shortcuts

Assign shortcut keys to plugin menu commands (see the Menu Extension section in Using Plugins) in this dialog box.

Dialog Box Options

Menu Items
Contains a list of all plugin menu commands with currently assigned shortcut keys.
Press new shortcut key, Assign
Focus the menu command to which you want to assign new shortcut key. Go to Press new shortcut key box and press that shortcut key. Click the Assign button to assign new shortcut key.
Clear
Focus the menu command to which you want to clear shortcut key. Click the Clear button to clear assigned shortcut key.
Reset
Restores original assignment of shortcut keys for all menu commands of this plugin.
