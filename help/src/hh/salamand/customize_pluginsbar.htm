Customizing Plugin Bar

Use command Plugins/Plugins Manager to open the Plugins Manager dialog box. Find plugin whose icon you want to add or remove from Plugin bar and focus its name in Installed plugins list. Use the Show PLUGIN-NAME icon in Plugin bar check box (where PLUGIN-NAME is name of plugin).
