Open Salamander: Appendix: File and Directory Concepts

Archive Path Specifier

Archive paths are similar to disk paths. The only difference is that each archive path consists of path to archive file and path in archive file (e.g. if C:\DIR\ARCHIVE.ZIP is an archive file containing PACKED-DIR directory, C:\DIR\ARCHIVE.ZIP\PACKED-DIR is an archive path specifying this directory and C:\DIR\ARCHIVE.ZIP\ is an archive path specifying root of the archive).
