Packing Files and Directories

This command is intended to pack selected files and directories into the archive. When you pack a directory, all the files and subdirectories in it are packed.

To pack files and directories:

  1. Select the files and directories you want to pack.
  2. Open the Pack dialog box:
    Menu:Files/Pack...
    Shortcut key:Alt+F5
    Toolbar:Choose the Pack button on the Top, Middle, or Bottom toolbar.
  3. In the Pack to archive box, type the name of the archive you want to create. If you type a filename without including a path, the archive will appear in the current directory.
  4. In the Packer box, select the type of archive you want to create.
  5. Choose the OK button.
