Open Salamander: Basic Working with Files and Directories

Selecting Files and Directories

Almost all file and directory operations (e.g. copying, moving, and deleting) work with chosen subset of files and directories from active (source) panel. We talk about this subset as about selected files and directories.

The list of files and directories in the working area of panel can contain three types of items - focused item, selected items, and normal items (see Working Area of Panel for details). When we talk about selected files and directories, we mean:

  • all selected items (if any) in the list,
  • or the focused item, if there are no selected items.

We will present here only ways of selecting items in the list (ways of focusing item are described in Focusing Item).

Selecting Files and Directories Using the Keyboard

Function Shortcuts Note
Select or unselect focused item Insert or Space Changes selection of focused item (from unselected to selected and vice versa). The Space key also calculates directory size if it is used to select directory.
Select all Edit/Select All (Ctrl+A or Ctrl+Num Plus)
Unselect all Edit/Unselect All (Ctrl+D or Ctrl+Num Minus)
Select using masks Edit/Select (Num Plus or Ctrl+=) Uses Select dialog box. Works with files and optionally also with directories, see check box Include directories to select/unselect operations in Configuration/Panels. For details on masks see File Masks.
Unselect using masks Edit/Unselect (Num Minus or Ctrl+-) Uses Unselect dialog box. Works with files and optionally also with directories, see check box Include directories to select/unselect operations in Configuration/Panels. For details on masks see File Masks.
Invert selection Edit/Invert Selection (Num Multiply) or Ctrl+Num Multiply The Ctrl+Num Multiply key works with files and directories. The Num Multiply key works with files and optionally also with directories, see check box Include directories to select/unselect operations in Configuration/Panels.
Restore Selection Edit/Restore Selection (Ctrl+W) Restores selection used for previous operation (selection is usually canceled when operation starts). It allows you to easily use another operation for the same files and directories.
Incremental selecting and unselecting Shift+Arrows, Shift+Home, Shift+End, Shift+PageUp, Shift+PageDown Selects or unselects items as you change (move) focused item - it works with all items between original and new focused items. It selects items if the original focused item is unselected and unselects items otherwise. To understand what the keys do, forget their Shift modifier and see Focusing Item.
Select all with the same extension Edit/Advanced/Select Same Extensions (Shift+Num Plus) Name parts can differ.
Unselect all with the same extension Edit/Advanced/Unselect Same Extensions (Shift+Num Minus) Name parts can differ.
Select all with the same name part Edit/Advanced/Select Same Names (Ctrl+Shift+Num Plus) Extensions can differ.
Unselect all with the same name part Edit/Advanced/Unselect Same Names (Ctrl+Shift+Num Minus) Extensions can differ.

Selecting Files and Directories Using a Mouse

There are two modes of using mouse in Open Salamander - classic and single-click modes, see check box Single-click to open an item in Configuration/Appearance. We present three tables with functions - for both modes, for classic mode, and for single-click mode.

Functions available in both modes of using mouse:

Function Shortcuts Note
Select or unselect items in the box Drag or Right Drag optionally with Shift or Ctrl modifier keys Selects or unselects items as you drag the box over them. To start this action, you must press (and hold) left or right (no difference) mouse button behind items in the working area of panel. It defines the first corner of the box (the opposite one is dragged by mouse pointer). If you start dragging without any modifier key, the current selection will be canceled. If you press (and hold) the Shift or Ctrl modifier key, the current selection will be kept. With the Shift modifier key all items in the box will be selected. With the Ctrl modifier key all selected items in the box will be unselected and vice versa.
Incremental selecting and unselecting Ctrl+Right Drag or Right Drag Selects or unselects items as you drag mouse pointer over them. To start this action, you must press (and hold) right mouse button (and the Ctrl modifier key) on item. It selects items if this item is unselected and unselects items otherwise. Use the Ctrl modifier key if you have selected check box Right-click to open shortcut menu in Configuration/Panels. If you place mouse pointer behind the border of working area of panel, the list of items in panel will scroll and so allow you to select/unselect also items which are currently not visible.

Functions available in classic mode of using mouse:

Function Shortcuts Note
Select or unselect clicked item Ctrl+Click Changes selection of clicked item (from unselected to selected and vice versa).
Select or unselect interval of items Shift+Click Selects or unselects items when you click item which is not focused - it works with all items between original and new focused items. It selects items if the original focused item is unselected and unselects items otherwise.

Functions available in single-click mode of using mouse:

Function Shortcuts Note
Select or unselect item Ctrl+rest pointer on item Changes selection of item on which you rest mouse pointer (from unselected to selected and vice versa). It works only on item that is not under mouse pointer (selecting/unselecting must be preceded by moving mouse pointer to the item).
Select or unselect interval of items Shift+rest pointer on item Selects or unselects items when you rest mouse pointer on item which is not focused - it works with all items between original and new focused items. It selects items if the original focused item is unselected and unselects items otherwise.
