Open Salamander: Dialog Boxes

Connect to Network Path

Use this dialog to connect to network path.

Dialog Box Options

User name
Type the account user name.

Note To connect using an account in a different domain, in the User Name box, type the domain name followed by a backslash (\) and the user name for the account, such as: Sales\cristal.

Password
Type the account password.
