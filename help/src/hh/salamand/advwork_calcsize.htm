Calculating Directory Sizes

This command allows you to calculate total sizes of files and subdirectories contained in the directories. This command applies to all directories in the active panel. Current selection is ignored. Calculated sizes will be displayed in the Size column in the panel. It may be handy if you need to search biggest subdirectories. When sizes are calculated, you can sort the panel contents by Size.

Use the Space key for selecting and calculating size of single directory.

To calculate directory sizes:

Menu:Commands/Calculate Directory Sizes
Shortcut key:Ctrl+Shift+F10
Toolbar:Choose the Calculate Directory Sizes button on the Bottom toolbar.
