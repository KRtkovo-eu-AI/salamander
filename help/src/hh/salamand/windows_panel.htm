Open Salamander: Window Components

Panel Components

The Panel includes the following:

  • Tab Strip
  • Directory Line
  • Header Line
  • Working Area
  • Information Line

Panel displays a collection of items (files and directories). Panel can display content in six different types of views. See Selecting Current View for details.

Tab Strip

When tabbed panels are enabled the panel is preceded by a tab strip. Each tab hosts its own directory, view state, and selection, so you can keep multiple locations open on the same side. Tabs can be reordered with the mouse, color-coded, assigned a custom prefix, duplicated to either side, or moved between panels using the commands in the Left/Right > Tabs submenus or the tab context menu.

The tab strip is shown only when at least one tab exists on that side. Disabling tabbed panels or closing additional tabs automatically hides the strip and restores the classic single-panel layout.
