Open Salamander: Glossary

L

list box

In a dialog box, a type of box that lists available choices — for example, a list of all files in a directory. If all the choices do not fit in the list box, there is a scroll bar.

local path

Start with the drive letter, followed by a colon (:) and a backslash (\), and separated directories with a backslash. See also universal naming convention (UNC).

long name

A directory name or filename longer than the 8.3 filename standard (up to eight characters followed by a period and a three-character extension) of the FAT file system. Windows automatically translates long names of files and folders to 8.3 names for MS-DOS users.
