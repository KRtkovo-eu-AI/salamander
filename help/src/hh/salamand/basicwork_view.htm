Open Salamander: Basic Working with Files and Directories

Viewing File

This command is intended to display file or document content. Distinct viewers can be associated with different file extensions (e.g. text viewer for *.txt and bitmap viewer for *.bmp). See Viewers configuration. Two distinct viewers can be associated with one extension - Viewer and Alternate Viewer (e.g. you can view *.htm file either in Internet Explorer Viewer or in Internal Viewer).

To view a file content:

  1. Focus the file you want to view.
  2. Choose View or Alternate View command:
    Menu:Files/View or Files/Alternate View
    Shortcut key:F3 or Alt+F3
    Toolbar:Choose the View button on the Top, Middle, or Bottom toolbar.

To view a file content using not associated viewer:

  1. Focus the file you want to view.
  2. Choose a viewer from View With submenu (it contains all viewers from Viewers configuration):
    Menu:Files/View With
    Shortcut key:Ctrl+Shift+F3
