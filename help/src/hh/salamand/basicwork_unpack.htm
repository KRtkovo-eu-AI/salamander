Open Salamander: Basic Working with Files and Directories

Unpacking Archive File

Open Salamander allows you to extract files and directories from an archive file. When you choose Unpack command, the files are extracted from the focused archive file and copied into the destination directory.

To unpack archive file:

  1. Focus the archive file you want to extract.
  2. Open the Unpack dialog box:
    Menu:Files/Unpack...
    Shortcut key:Alt+F9 or Alt+F6
    Toolbar:Choose the Unpack button on the Top, Middle, or Bottom toolbar.
  3. In the Unpack archive to box, specify the destination drive or directory.
    If you type dot (.), the extracted files and directories will appear in the current directory.
  4. Choose the OK button.
