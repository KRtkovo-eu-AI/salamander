Open Salamander: Working with Windows and Panels

Focusing Item

The panel can contain any number of items (files and directories) in a list. If the list contains some items, one of these items is focused. The focused item is highlighted in the active panel (see Working Area of Panel for details). The focused item in inactive panel is not highlighted, but when the panel becomes active, it is highlighted.

You can change focus in the list of items (choose another item to be focused). It can be done using mouse or keyboard. With mouse you can just click the item you want to focus. The keyboard shortcuts are summarized in the following table.

The Keyboard Shortcuts:

Use To
Arrows Focus item in the specified direction from the currently focused item.
Home Focus the first item.
End Focus the last item.
Ctrl+Home Focus the first file.
Page Up Focus the item up one page from the currently focused item.
Page Down Focus the item down one page from the currently focused item.
Alt+Down arrow key Focus next selected item (from the currently focused item).
Alt+Up arrow key Focus previous selected item (from the currently focused item).
