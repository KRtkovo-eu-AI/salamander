Open Salamander: Working with Windows and Panels

Common Shortcut Keys

The following table lists the common shortcut keys.

Key Meaning
Alt+Escape Switches to the next application.
Alt+Tab Switches to the next application.
Alt+Shift+Tab Switches to the previous application. The user must press and hold down Alt+Shift while pressing Tab.
Alt+F4 Closes an application or a window.
Alt+Space Opens the window menu for the application's main window.
F10 Go to menu mode.
Ctrl+Escape Switches to the Start menu.
Print Screen Copies an image on the screen onto the clipboard.
Alt+Print Screen Copies an image in the active window onto the clipboard.
