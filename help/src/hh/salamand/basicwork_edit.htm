Open Salamander: Basic Working with Files and Directories

Editing File

This command is intended to edit file or document content. Distinct editors can be associated with different file extensions (e.g. text editor for *.txt and bitmap editor for *.bmp). See Editors configuration.

To edit a file content:

  1. Focus the file you want to edit.
  2. Choose Edit command:
    Menu:Files/Edit
    Shortcut key:F4
    Toolbar:Choose the Edit button on the Top, Middle, or Bottom toolbar.

To edit a file content using not associated editor:

  1. Focus the file you want to edit.
  2. Choose an editor from Edit With submenu (it contains all editors from Editors configuration):
    Menu:Files/Edit With
    Shortcut key:Ctrl+Shift+F4

To create and edit a new file:

  1. Open the directory in which you want the new file to appear.
  2. Open the Edit New File dialog box:
    Menu:Files/Edit New File
    Shortcut key:Shift+F4
    Toolbar:Choose the Edit New File button on the Top, Middle, or Bottom toolbar.
