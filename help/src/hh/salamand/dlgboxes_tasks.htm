Open Salamander: Dialog Boxes

Open Salamander Task List

Use this dialog to break or terminate a not-responding Open Salamander task.

Dialog Box Options

List of running Open Salamander tasks
Contains a list of all Open Salamander tasks. Current task has "current process" appendix.
Highlight
Flash main window of selected Open Salamander task.
Break
Breaks selected Open Salamander task and invokes its Bug Report dialog.
Terminate
Terminates selected Open Salamander task.
Refresh
Refresh list of running Open Salamander tasks
