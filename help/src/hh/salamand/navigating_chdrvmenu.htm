Open Salamander: Navigating Through Directories

Change Drive Menu

Use Change Drive commands from Left and Right menus to open the Change Drive menu. The Change Drive menu allows you to change the current directory in the panel. The Change Drive menu contains:
  • local drives (floppy disk drives, CD-ROM drives, hard disk drives, etc.),
  • network drives,
  • opened plugin file systems,
  • the Documents item,
  • the Network item,
  • the list of plugin file systems,
  • the As Other Panel item,
  • and the list of Hot Paths.

Local Drives and Network Drives

Use one of these items to change the current directory to the last visited (or root) directory on chosen drive. You can use drive letters as keyboard shortcuts for drives in menu (these letters are underlined in menu). Note that menu contains types of floppy disk drives (5.25", 3.5", or 8"), labels of hard disk drives and CD-ROM drives, free space on hard disk drives, and network paths of network drives. You can select which drives are visible and separate groups of drives, see Configuration/Change Drive Menu and Drive Bar.

Opened Plugin File Systems

Use one of these items to change the current directory to the last visited directory in chosen opened plugin file system (e.g. connections to some FTP servers or SCP/SFTP servers). Opened plugin file system from other panel is disabled because it cannot be opened also in active panel at the same time.

The Documents Item

Use this item to change the current directory to the Documents folder. Note that this item is optional - see Documents check box in Configuration/Change Drive Menu and Drive Bar.

The Network Item

Use this item to change the current directory to a shared directory on a computer on the network. Note that this item is optional - see the Network check box in Configuration/Change Drive Menu and Drive Bar.

The List of Plugin File Systems

Use one of these items to open plugin file system in panel. If you don't find some plugin file system in this list, find appropriate plugin in the Plugins Manager dialog box and set the Show FS item in Change Drive menu and Drive bar check box. See Opening Plugin File Systems for details.

The As Other Panel Item

Use this item to change the current directory to the current directory of other panel. Note that this item is optional - see As Other Panel check box in Configuration/Change Drive Menu and Drive Bar.

The List of Hot Paths

Use one of these items to change the current directory to the chosen Hot Path. For details on Hot Paths, see Using Hot Paths.

Location

Menu:Left/Change Drive for the left panel and Right/Change Drive for the right panel.
Shortcut key:The Alt+F1 key for the left panel and the Alt+F2 key for the right panel.
Toolbar:Choose the Change Drive button on the Panel toolbar contained in the Directory line of the panel.
