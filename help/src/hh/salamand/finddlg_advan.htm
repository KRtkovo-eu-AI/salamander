Open Salamander: Dialog Boxes: Find

Advanced Options

Use this dialog box to specify advanced search options.

Dialog Box Options

Attributes
Each attribute is represented by one check box with the following meaning:
Items with this attribute will be found.
Items without this attribute will be found.
Ignore this attribute.
File size
Allows you to specify the range for the size of the file you're looking for.
Date and Time
Allows you to find files and directories that were modified in specified time period.
Reset
Sets all advanced options to be ignored.
