Open Salamander: Basic Working with Files and Directories

Creating a Directory

You can create a directory by using the Create Directory command. After you create a directory, you can copy files and directories to it from other locations by using the Move/Rename and Copy commands.

To create a directory:

  1. In the panel open the directory in which you want the new directory to appear.
  2. Open the Create Directory dialog box:
    Menu:Commands/Create Directory...
    Shortcut key:F7
    Toolbar:Choose the Create Directory button on the Top, Middle, or Bottom toolbar.
  3. In the Directory Name box, type a name for the new directory. You can also create more directories at once (e.g. if you enter C:\a\b\c and C:\a does not exist, three directories will be created).
  4. Choose the OK button.
