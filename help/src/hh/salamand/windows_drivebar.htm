Open Salamander: Window Components

Drive Bar

The Drive bar contains all available local and network (mapped) drives, the Documents item, the Network item, and the list of plugin file systems. You can choose if you want to display the Drive bar only for the active panel or for both panels (see Two Drive Bars item in Options/Show menu). You can drag the Drive bar to another location.

Local Drives and Network Drives

Click on one of these items to change the current directory to the last visited (or root) directory on chosen drive. Rest mouse pointer on a drive button to see drive information (the volume label, the drive type, the free space on the drive, and the network path in UNC format). Right-click on a drive button to see context menu for the drive root directory. You can select which drives are visible and separate groups of drives, see Configuration/Change Drive Menu and Drive Bar.

The Documents Item

Click on this item to change the current directory to the Documents folder. Note that this item is optional - see Documents check box in Configuration/Change Drive Menu and Drive Bar.

The Network Item

Click on this item to change the current directory to a shared directory on a computer on the network. Note that this item is optional - see the Network check box in Configuration/Change Drive Menu and Drive Bar.

The List of Plugin File Systems

Use one of these items to open plugin file system in panel. If you don't find some plugin file system in this list, find appropriate plugin in the Plugins Manager dialog box and set the Show FS item in Change Drive menu and Drive bar check box. See Opening Plugin File Systems for details.
