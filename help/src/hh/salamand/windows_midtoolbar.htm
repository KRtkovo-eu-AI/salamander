Open Salamander: Window Components

Middle Toolbar

The Middle toolbar contains the most commonly used menu commands.

You can:
  • Click a toolbar button once to carry out the action represented by that button.
  • Rest mouse pointer on a toolbar button to see description of the button and hotkey (if any).
  • Right-click on the toolbar and then click Customize Toolbar to customize toolbar buttons.
  • Drag the toolbar to another location.
