Open Salamander: Dialog Boxes: Configuration

External Archivers Locations

Allows you to assign external archiver file name to the variables used in Open Salamander archivers configuration.

Dialog Box Options

External Archivers
Displays a list of external archivers supported by Open Salamander.
List and Unpack
Displays the external archiver file name assigned to the selected variable in External Archivers. This archiver will be used for listing and unpacking content of archive.
Delete and Pack
Displays the external archiver file name assigned to the selected variable in External Archivers. This archiver will be used for deleting files from archive and packing files into archive. If selected archiver supports all operations (listing, unpacking, deleting, and packing), Delete and Pack box will be disabled. Otherwise different file name of external archiver must be assigned.

Note

Use Options/Archivers Autoconfiguration to search for installed external archivers on your machine.
