Open Salamander: Introduction

Technical Support

If you experience problems while using Open Salamander, our technical support is always ready to help you.

Before You Contact Us

Since we have many users we ask that you do the following before contacting us:

  • Check whether you have the most recent version of Open Salamander.
  • Check to see that the information you need is not already available in the Help file.

Reaching Technical Support

For technical support you can use Open Salamander's Official Support Forum (forum.altap.cz) or if you need to send us sensitive data, use email support@altap.cz.
