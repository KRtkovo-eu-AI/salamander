Open Salamander: Dialog Boxes

Archivers Autoconfiguration

Use this dialog box to find external archivers on you drives.

Dialog Box Options

Available archivers
Displays a list of the packers that are to be searched on your machine.
Add/Remove custom (un)packers
When selected, not only archives association in panels will be modified, but also packers will be removed from and added to the list of packers in the pack and unpack dialog boxes. When not selected, this list of custom archivers remains unmodified.
Drives
Opens a dialog allowing to specify which drives will be searched for archivers.
Start
Begin searching of external archivers.
