Open Salamander: Securing Files and Directories

Setting Files and Directories Permissions

You can set directory and file permissions only on drives formatted to use the Windows NT file system (NTFS). Only files or directories permissions can be set at one time.

To change permissions:

  1. Select the files or directories which permissions you want to change.
  2. Choose the Permissions command:
    Menu:Files/Security
    Shortcut key:Ctrl+P
    Toolbar:Choose the Security button on the Top or Middle toolbar.
