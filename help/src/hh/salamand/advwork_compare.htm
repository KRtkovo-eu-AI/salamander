Comparing Directories

This command allows you to compare contents of current directories in the left and the right panel. After comparing, files and directories that are not present in the other panel, files that are newer or bigger, files with different attributes or content, and directories containing different files or directories will be selected in both panels.

To compare directories contents:

  1. Open the directories you want to compare in the left and right panel.
  2. Open the Compare Directories dialog box:
    Menu:Commands/Compare Directories
    Shortcut key:Ctrl+F10
    Toolbar:Choose the Compare Directories button on the Top, Middle, or Bottom toolbar.
  3. Select compare criteria, see the Compare Directories dialog box for details.
  4. Choose the OK button.
