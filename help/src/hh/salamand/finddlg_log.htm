Open Salamander: Dialog Boxes: Find

Find Log

All errors which occur during searching for files and directories and all ignored directories are displayed in this dialog box.

Dialog Box Options

Messages generated during searching
Contains a list of errors and ignored directories.
Focus
Use this button to focus the file or directory connected with the error or the ignored directory in the active panel.
Ignore
Use this button to add the directory connected with the error to the list of ignored directories (see the Ignore Directory List dialog box for details).
