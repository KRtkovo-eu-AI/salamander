Open Salamander: Dialog Boxes: Configuration

Packers in Pack Dialog Box

Allows you to manage packers used for Pack command.

Dialog Box Options

Name
Displays a list of all packers used for Pack command. You can reorder names according to your most often used packers.
Pack with
Packer used for selected packer name. It can be either external program or plugin.
Default extension
Extension used when new archive is created with Pack command.
Pack to archive
When external program is used for packing, Command specifies file name of used external archiver. Using of variable from External Archivers Locations is recommended when packer is known in Open Salamander. Arguments are parameters passed to the external program.
Pack to archive and then delete from disk
When external program is used for packing and deleting files may be supported, Command specifies file name of used external archiver. Using of variable from External Archivers Locations is recommended. Arguments are parameters passed to the external program.
Support long names
Check this check box when external archiver supports long file names. Otherwise clear this check box and Open Salamander prepares MS-DOS 8.3 file names.
Need List of Files in ANSI Charset
Check this check box if external program used for packing and deleting files requires list of files in ANSI charset (some Win32 archivers). Clear this box if it requires list of files in OEM charset.
