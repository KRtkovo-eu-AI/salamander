Making List of Files and Directories

This command allows you to save the list of selected files and directories into the text file, to the clipboard, or to open it in the Internal Viewer.

To make the list:

  1. Select the files and directories you want to list.
  2. Open the Make File List dialog box:
    Menu:Commands/Make File List...
    Shortcut key:Ctrl+M
  3. Specify the format of the list in the Line field.
  4. Choose the destination for the resulting list. If you want to save it to the file, enter its filename to the File field. Moreover you can specify what to do if the file already exists - if to overwrite it or append the list to its end.
  5. Choose the OK button.
