Converting Contents of Text Files

This command allows you to convert the contents of text files including change of character coding and end of line characters.

To convert the contents of files:

  1. Select the files and directories you want to convert.
  2. Open the Convert dialog box:
    Menu:Files/Convert...
    Shortcut key:Ctrl+K
    Toolbar:Choose the Convert button on the Top or Middle toolbar.
  3. Use file masks to specify what files should be converted.
  4. Select the current and required character coding and requested end of lines for converted files.
  5. Specify if to include files contained in selected directories.
  6. Choose the OK button.
