Compressing and Encrypting of Files and Directories

You can compress or encrypt files and directories on NTFS volumes (see NTFS section in File Systems). It's not possible to combine compressing and encrypting. So if you compress encrypted file, the result will be compressed but not encrypted file (and vice versa). Use the following commands:

  • NTFS Compress (it is also on Top and Middle toolbars): compress selected files and directories,
  • NTFS Uncompress (it is also on Top and Middle toolbars): uncompress selected files and directories,
  • NTFS Encrypt: encrypt selected files and directories,
  • NTFS Decrypt: decrypt selected files and directories.

All described commands are available in menu Files/NTFS Commands. It's also possible to compress and encrypt files and directories by setting or clearing of Compressed and Encrypted attributes using the Change Attributes command.
