Open Salamander: Shortcuts

Keyboard Shortcuts

There are many hotkeys defined in Open Salamander. These can speed up your work with files and directories. You can find all the functions in menu together with associated hotkeys, so you can learn hotkeys just in time of use. Take this list as overview of all hotkeys, not as something you have to know to use Open Salamander efficiently.

Shortcut keys for plugin commands and windows are mentioned in section Plugins of this help (see Contents tab).
