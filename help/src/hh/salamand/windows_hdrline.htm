Header Line

Header line displays a heading above columns in the panel in detailed mode.
You can:
  • Click the Header line button to sort panel items according to clicked item. Click the item again to reverse the sort order. Only some of the columns can be sorted.
  • Drag the Header line divisions that separate header parts to set the width of each column. Drag operation is not available when column is in Automatic Column Width; the width of the column will fit the maximal size of the column entries.
  • Right-click on the Header line item then click the Automatic Column Width or Smart Column Mode (only on the Name column) to choose the mode of the appropriate column. See Configuration/Views for more information.
