Open Salamander: Dialog Boxes

Customize Toolbar

Use this dialog box to add, remove, and rearrange the buttons on the toolbar. Most of the commands available on Open Salamander menus has an equivalent button.

Dialog Box Options

Available toolbar buttons
Displays the buttons that you can add to the toolbar.
Current toolbar buttons
Displays the buttons in the order they appear (from left to right) on the toolbar.
Add ->
Moves the button selected in the Available toolbar buttons list to the Current toolbar buttons list.
<- Remove
Moves the button selected in the Current toolbar buttons list to the Available buttons list.
Close
Closes the dialog, accepting the changes you have made.
Reset
Restores the default set of buttons on the toolbar in the active window.
Move Up
Moves the selected button up one line. On the toolbar, the button shifts left one space.
Move Down
Moves the selected button down one line. On the toolbar, the button shifts right one space.
