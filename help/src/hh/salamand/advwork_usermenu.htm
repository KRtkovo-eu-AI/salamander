Using User Menu

User Menu allows you to execute external programs or open documents. See Customizing User Menu for details.

To open User Menu:

Menu:Commands/User Menu
Shortcut key:F9
Toolbar:Choose the User Menu button on the Top, Middle, or Bottom toolbar.
User Menu bar:Choose the required item on the User Menu bar. You can also drag & drop a file as an argument to the User Menu command (drop the file on the item on the User Menu bar).
