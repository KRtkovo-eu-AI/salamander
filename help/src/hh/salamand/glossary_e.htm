encryption

The process of making information indecipherable to protect it from unauthorized viewing or use, especially during transmission or when it is stored on a transportable magnetic medium.

environment variable

A string consisting of environment information, such as a drive, path, or filename, associated with a symbolic name that can be used by Windows. To define environment variables, use System in Control Panel (Windows NT) or use the Set command from the Windows command prompt.
