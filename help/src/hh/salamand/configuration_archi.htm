Open Salamander: Dialog Boxes: Configuration

Archivers

Allows you to change the behavior of the archivers.

Dialog Box Options

Use simple icons in archives
When checked, only the simple symbols instead of default icons provided by the system will be shown in archives.
Pack to other panel
When checked, default destination directory for created archive will be set to the target (inactive) panel instead of the source (active) panel.
Unpack to other panel
When checked, default destination directory for unpacked archive will be set to target (inactive) panel instead of the source (active) panel.
Unpack to subdirectory named by archive
When checked, default destination directory will be ended with the name of the unpacked archive; otherwise destination directory will be set only to active or inactive panel in accordance to the Unpack to other panel check box.
