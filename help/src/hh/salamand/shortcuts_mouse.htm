Mouse Shortcuts in Panels
Mouse Shortcuts in Internal Viewer

Panels

Do To
Rotate Mouse WheelScroll panel vertically or horizontally (if only horizontall scroll bar is visible)
Tilt Mouse WheelScroll panel horizontally
Ctrl+Rotate Mouse WheelSet thumbnail size in the Thumbnails mode
Shift+Rotate Mouse WheelScroll panel horizontally
Alt+Rotate Mouse WheelSelect current view

Internal Viewer

Do To
Rotate Mouse WheelScroll window vertically
Tilt Mouse WheelScroll window horizontally
Shift+Rotate Mouse WheelScroll window horizontally

Remarks

Mouse wheel tilting is supported from Windows Vista. Also mouse with support for wheel tilting must be used.
