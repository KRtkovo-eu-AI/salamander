Opening Properties of Files and Directories

This command allows you to open Properties dialog box for selected files and directories.

To open Properties dialog box:

  1. Select the files and directories whose properties you want to change.
  2. Open the Properties dialog box:
    Menu:Files/Properties
    Context menu:Right-click selected files and directories and select Properties.
    Shortcut key:Alt+Enter
    Toolbar:Choose the Properties button on the Top or Middle toolbar.
