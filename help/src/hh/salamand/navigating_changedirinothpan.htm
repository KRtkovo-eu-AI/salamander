Open Salamander: Navigating Through Directories

Changing Directory in Other Panel

You can change directory in other (inactive) panel to focused directory in active panel. You can also locate file focused in active panel in other (inactive) panel.

To change directory or locate file in other panel:

  1. Focus the directory or the file in active panel.
  2. Change directory or locate file in other panel:
    Shortcut key:Ctrl+Shift+Left/Right Arrow key - use the arrow in the direction to other panel to activate other panel (the arrow in opposite direction does not activate other panel)
    Toolbar:Choose the Focus Name in Other Panel button on the Top or Middle toolbar - it activates other panel.
    Choose the Open Name in Other Panel button on the Top or Middle toolbar - it does not activate other panel.
