Setting Visibility of Panel Components

You can customize the design of the left or right panel by setting visibility of its components (the Directory line, the Header line, and the Information line). If you want to know how panel components look like, see a screenshot with description in chapter Panel Components.

To set visibility of left panel components use three menu items in menu Left/Show. To set visibility of right panel components use three menu items in menu Right/Show.

When tabbed panels are enabled, each side also displays a tab strip above the panel whenever more than one tab exists on that side. The tab bar follows the tab state automatically, so it disappears when only the default tab remains or when tabbed panels are turned off.
