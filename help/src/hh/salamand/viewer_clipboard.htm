Open Salamander: Using Internal Viewer

Copying Text to Clipboard

You can copy selected text from viewer to the clipboard. If you are using some conversion (see Converting Text), text is copied after this conversion.

When Auto-Copy Selection to Clipboard in the Options menu in the viewer is checked and you select some text, the selected text will be automatically copied to the clipboard.

To copy selected text to the clipboard:

  1. Select text you want to copy to the clipboard. See Selecting Text for details.
  2. Copy selected text to the clipboard:
    Viewer menu:Edit/Copy
    Viewer shortcut key:Ctrl+C
    Viewer context menu:Copy
