Open Salamander: Dialog Boxes

Save Selection

Use this dialog box to save a current selection in the active panel.

Dialog Box Options

Save selected names to
Selection memory
The selection will be saved to the selection memory (only for use inside Salamander).
Clipboard
The selection will be saved to the clipboard (format is simple: one name per line, so you can use the selection also in text editors, etc.).
