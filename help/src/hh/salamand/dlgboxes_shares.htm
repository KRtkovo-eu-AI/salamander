Open Salamander: Dialog Boxes

Shared Directories

Use this dialog box to see a list of all shared directories, stop sharing, or focus shared directory in the active panel.

Dialog Box Options

List of shared directories
Displays a list of all shared directories (share name, shared path, and comment).
Stop Sharing
Use this button to stop sharing of focused shared directory.
Focus
Use this button to change directory in the active panel to the focused shared directory.
Refresh
Use this button to read all shared directories from system again. It is needed if you change sharing from other application while this dialog is opened.
