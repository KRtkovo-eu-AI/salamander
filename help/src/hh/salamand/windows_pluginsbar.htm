Open Salamander: Window Components

Plugin Bar

The Plugin bar contains plugins with the Menu Extension function (see the Menu Extension section in Using Plugins). Showing the plugin icon (button) in the Plugin bar is optional, see the Show plugin icon in Plugin bar check box in the Plugins Manager dialog box.

You can:
  • Click a plugin button to open the context menu with plugin commands.
  • Rest mouse pointer on a plugin button to see the name of the plugin.
  • Right-click on the bar and then click Customize Toolbar to open the Plugins Manager dialog box.
  • Drag the bar to another location.
