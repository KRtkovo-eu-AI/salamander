Finding Shortcut or Link Target

This command allows you to find a target of a focused shortcut (.lnk file) or link (junction point or symbolic link). If the shortcut or link target is a file, it will be focused in the current panel. If the shortcut or link target is a directory, the path in the current panel will change to this directory.

To find a shortcut or a link target:

  1. Focus a shortcut (.lnk file) or a link (it has icon with link overlay) in the panel.
  2. From the Commands menu select Go to Shortcut or Link Target (the Ctrl+T key) or click on the Go to Shortcut or Link Target button on the Top or Middle toolbar. For the shortcut of a directory you can also use the Enter key or double-click.
