Open Salamander: Working with Windows and Panels

Activating Command Line

The Command Line can be visible all the time or only when you activate it (see Setting Visibility of Window Components). If you want to know how the Command Line looks like, see a screenshot with description in chapter Main Window Components.

To activate the Command Line from the panel use the Ctrl+Tab key. To return back to the panel use the same key or the Escape key.

You can activate the Command Line also using a mouse. In this case the Command Line must be visible all the time. If it is, use click to activate it.
