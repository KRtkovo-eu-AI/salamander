Open Salamander: Using Internal Viewer

Drag & Drop

You can copy selected text from viewer to any application (e.g. Mozilla Firefox, Microsoft Word) using drag & drop. If you are using some conversion (see Converting Text), text is copied after this conversion.

To copy selected text using drag & drop:

  1. Select text you want to copy. See Selecting Text for details.
  2. Move mouse pointer to a selected text, press (and hold) the left mouse button, drag mouse pointer to a place where you want to copy selected text to, and then release mouse button to finish operation.
