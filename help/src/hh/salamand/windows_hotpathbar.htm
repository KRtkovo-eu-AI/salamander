Open Salamander: Window Components

Hot Path Bar

The Hot Path bar contains all defined Hot Paths, see Using Hot Paths for details.

You can:
  • Click a Hot Path button to change directory in the active panel to the Hot Path.
  • Rest mouse pointer on a Hot Path button to see its path.
  • Right-click on the bar and then click Customize Toolbar to open the Configuration/Hot Paths dialog box.
  • Drag the bar to another location.
