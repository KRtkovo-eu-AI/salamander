Open Salamander: Window Components

Bottom Toolbar

The Bottom toolbar contains buttons that are corresponds to F1 to F12 keys on the keyboard. Buttons on the Bottom Toolbar have different functions when you use them in combination with keys such as Ctrl, Alt, and Shift on a keyboard — just as the keys on the keyboard have different functions when you press them in combination with Ctrl, Alt, or Shift.

You can:
  • Click a toolbar button once to carry out the action represented by that button.
  • Press the Ctrl, Alt, and Shift keys to modify toolbar context.
  • Rest mouse pointer on a toolbar button to see description of the button.
