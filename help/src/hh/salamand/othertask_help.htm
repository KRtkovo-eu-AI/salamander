Accessing Help

There are many ways to access Help for answers to questions about Open Salamander and for information about keywords. Descriptions of a few of the most common and useful ways of accessing Help follows.

  • Press F1 to display help on a topic associated with the currently opened window or dialog box.
  • Press Shift+F1 in Open Salamander's main window. The mouse cursor changes to the help cursor, and you can select any visible object in the user interface, such as a menu item, toolbar button, or window. This opens Help on a topic that describes the selected item.
  • Open Help menu from Open Salamander main window. Select on of the following items: Contents, Index, Search, and Keyboard Shortcuts.
  • Click Help button in dialog box to display help on a topic associated with the currently opened dialog box.

Tips

To switch between your application and Help window, press Alt+Tab or use the Windows taskbar. If you leave Help active, you can immediately return to the last topic you were viewing.
