Open Salamander: Working with Windows and Panels

Setting Window Always on Top

If you want to have the main window of Open Salamander always above nearly all other application's windows, select the menu item Always on Top from main window's window menu (click on icon in upper left corner of Open Salamander's main window or use the Alt+Space key). When this item is selected, the main window becomes a top-level window and can be overlapped only by other top-level windows.

To toggle Window Always on Top option:

Menu:Options/Always on Top
Window menu:Always on Top
