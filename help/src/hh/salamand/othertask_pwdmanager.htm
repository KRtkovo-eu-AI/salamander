Using Password Manager

Open Salamander can protect sensitive information like FTP or WinSCP passwords by strong AES cipher. In order to protect such sensitive information you need to set a Master Password. When a Master Password is not set, stored passwords can be easily displayed in dialog boxes or decrypted by malicious software that have infiltrated your computer.

If you set the Master Password, each new instance of Open Salamander will ask you to enter the Master Password the first time it needs to access protected information. Later you can change or remove the Master Password. If you remove the Master Password, sensitive information will not be protected anymore.

To set the Master Password:

  1. Open Options/Configuration/Security dialog box.
  2. Set the Use Master Password option and choose your Master Password. IMPORTANT: If you forget your Master Password, we will not be able to recover it, all protected sensitive information like FTP or WinSCP passwords will be lost.

To change the Master Password:

  1. Open Options/Configuration/Security dialog box.
  2. Click the Change Master Password button.
  3. Enter current and new Master Password.

To remove the Master Password:

  1. Open Options/Configuration/Security dialog box.
  2. Clear the Use Master Password option and enter current Master Password.

To enter the Master Password:

  1. Open Salamander will ask you to enter the Master Password when it needs to access protected information.

Note: Avoid weak Master Passwords. Good passwords have at least six characters and contain both lowercase and uppercase letters, numbers and special characters, such as delimiters, symbols, letters with accent, etc.
