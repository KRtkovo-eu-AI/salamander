Using Open Salamander Task List

Open Salamander Task List is intended to control execution of all running Open Salamander tasks (version 2.52 or later). The most important feature is possibility to break a not-responding task (do it from some responding Open Salamander task). The break should invoke the Bug Report dialog in a target task, so you can send us the Bug Report about "hung" Open Salamander. This should help us to remove such bugs.

To break a not-responding (frozen) Open Salamander task:

  1. Start a new Open Salamander task.
  2. Open the Open Salamander Task List dialog box: Help/Task List.
  3. In the List of running Open Salamander tasks box select the not-responding task.

    Note Current task has "current process" appendix.

  4. Choose the Break button.
  5. The break should invoke the Bug Report dialog in the selected task.
  6. Activate the Bug Report dialog box and send us the Bug Report about "hung" Open Salamander.
