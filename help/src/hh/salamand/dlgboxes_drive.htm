Open Salamander: Dialog Boxes

Drive Information

Use this dialog box to display information about drive or to change volume name.

Dialog Box Options

Volume Name
Displays the name of the volume. You can enter another name to set new one. Labels on NTFS disks can contain up to 32 characters. Labels on FAT and HPFS disks can contain up to 11 characters.
