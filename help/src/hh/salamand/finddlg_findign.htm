Open Salamander: Dialog Boxes: Find

Ignore Directory List

Use this dialog box to specify directories which should be ignored when searching for files and directories. Directories in which you start the search cannot be ignored.

Dialog Box Options

Directories to skip during search
Contains a list of ignored directories. You can enter a relative name of directory (e.g. Local Settings\Temporary Internet Files), name of directory in the root path (e.g. \System Volume Information), or absolute path to directory (e.g. C:\TEMP). To add new item focus the empty item (at the bottom) and start typing or press the Insert key or click the New button in the toolbar. To modify item click on the focused item or press the F2 key or click the Modify button in the toolbar. To delete item press the Delete key or click the Delete button in the toolbar.
Reset
Sets ignored directories to default values.
