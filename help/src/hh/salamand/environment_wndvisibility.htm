Open Salamander: Working with Windows and Panels

Setting Visibility of Window Components

The design of Open Salamander can be customized by setting visibility of window components: the Top toolbar, the Plugin bar, the User Menu bar, the Hot Path bar, the Drive bar, the two Drive bars, the Middle toolbar, the Command line, and the Bottom toolbar. You can also lock the toolbars and show/hide text labels in the User Menu bar. If you want to know how window components look like, see a screenshot with description in chapter Main Window Components.

All described commands can be found in menu Options/Show.
