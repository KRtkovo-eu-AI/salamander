Open Salamander: Dialog Boxes

Convert

Use this dialog box to change coding or to convert end of line (EOL) characters of selected files.

Dialog Box Options

Convert files
Use file masks to specify what files should be converted.
Change Coding
Select current and required coding of converted files.
Change end of line (EOF) characters
Allows you to change end of line (EOF) characters in selected files.
Include subdirectories
When checked, operation includes files contained within selected directories.
