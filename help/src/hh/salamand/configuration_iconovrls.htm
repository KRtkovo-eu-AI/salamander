Open Salamander: Dialog Boxes: Configuration

Icon Overlays

Specifies which icon overlays are displayed in panels. Icon overlays are small images placed at the lower-left corner of the file icon (e.g. small arrow which represents that file is a shortcut).

Dialog Box Options

Display custom icon overlays (provided by shell extensions)
When checked, all checked icon overlay handlers from the Custom Icon Overlay Handlers list are used (their icon overlays are displayed in panels). Otherwise, no icon overlay handler is used.
Custom Icon Overlay Handlers
Contains a list of all available custom icon overlay handlers. Clear check boxes of handlers which should not be used (their icon overlays will not be displayed in panels). This list is disabled when the Display custom icon overlays box is not checked.
