Open Salamander: Window Components

User Menu Bar

The User Menu bar contains items from User Menu. User Menu bar is intended for quick access to these commands. Only icon, or icons and text labels can be displayed.

You can:
  • Click a toolbar button once to carry out the action represented by corresponding menu item.
  • Drag the toolbar to another location.
  • Drag file to the User Menu bar to add new item.
  • Drag file or directory to the User Menu bar button to execute this command with dragged parameter.
  • Right-click on the toolbar and then click Customize Toolbar to customize toolbar buttons or click Text labels to switch displaying of text labels behind icons.
