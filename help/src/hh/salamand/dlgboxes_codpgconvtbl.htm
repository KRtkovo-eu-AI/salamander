Code Page Conversion Tables

Choose the code page conversion tables in this dialog box. These tables are used in the Internal Viewer, the Convert command, and in some plugin viewers. It allows you to convert text stored in different than ANSI character coding (e.g. EBCDIC, MAC, or MS-DOS) to ANSI coding which can be properly displayed in Windows.
