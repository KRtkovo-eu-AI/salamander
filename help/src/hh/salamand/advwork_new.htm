Open Salamander: Advanced Working with Files and Directories

Creating New Object

This command allows you to create a new object (folder, shortcut, various document files, etc). Open Salamander set focus on just created object and you can easily work with it, e.g. open it.

To create a new object:

  1. Open the directory in which you want the new object to appear.
  2. From Files menu select New and choose object type you want to create.
