Opening System Folder

You can open special folders like Computer, Recycle Bin, Control Panel, or Network directly from Salamander using the submenu Open Folder from the menu Commands.

You can also open current directory in the Windows Explorer. Use the Shift+F3 key, the Commands/Open Folder/Active Folder command, or the Open Active Folder button on the Top or Middle toolbar.
