Open Salamander: Dialog Boxes

Change Case

Use this dialog box to change name case of selected files and directories.

Dialog Box Options

Change Case to
Select one of following name case types.
Name Case Type From To
Lower case mY docuMENT.tXt my document.txt
Upper case mY docuMENT.tXt MY DOCUMENT.TXT
Partially mixed case (name in mixed, extension in lower) mY docuMENT.tXt My Document.txt
Mixed case mY docuMENT.tXt My Document.Txt
Change
Select which parts of file name should be changed.
Include subdirectories
When checked, operation includes files and directories contained within selected directories.
