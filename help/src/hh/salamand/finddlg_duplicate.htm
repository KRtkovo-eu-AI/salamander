Find Duplicate Files

Use this dialog box to specify what criteria must two files fulfill to be considered as duplicates. It can be any combination of following criteria: the same name, the same size, and the same content.

Dialog Box Options

Find files with the same
Name
When checked, names of two compared files must be the same. Names are compared case insensitively, so lower and upper case characters are equal.
Size
When checked, sizes of two compared files must be the same.
Content
When checked, contents of two compared files must be the same.
