Opening Command Shell

You can open command shell window for current directory.

To open the command shell window:

Menu:Commands/Command Shell
Shortcut key:Num / or Ctrl+/
Toolbar:Choose the Command Shell button on the Top or Middle toolbar.

Remarks

UNC paths are not supported in the command shell. If you open command shell on the UNC path, Windows directory will be used as current path.
