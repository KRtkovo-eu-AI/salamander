Highlighting Groups of Files and Directories

When you are browsing directories, you may appreciate if some files and directories are displayed in different colors (highlighted). For example, you can highlight all ZIP archives (*.zip files).

To create a rule for highlighting:

  1. In the Options menu, select Configuration, and then choose the Colors dialog box.
  2. In the Masks Priority Lists select the last empty row.
  3. Type a filename or use file masks to specify a group of files and directories you want to highlight.
  4. Press the Enter key.
  5. Select or clear the check boxes for the attributes of the filename or group you want to highlight.

    Keep the check box in the third state if the attribute is not significant for you.

  6. Use buttons on the right side to choose colors for the highlight.
  7. Choose the OK button.
