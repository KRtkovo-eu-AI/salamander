Open Salamander: Appendix: File and Directory Concepts

Wildcard Characters

Asterisk (*) and question mark (?) are wildcard characters. The asterisk matches any sequence of characters, whereas the question mark matches any single character. You can use these characters to build masks matching groups of file and directory names.

To Search ForUse Mask
Names with any number of characters in the name part and doc extension. *.doc
Names with the name part readme and any number of characters in the extension. readme.*
Names with the name part readme and any three-character extension. readme.???
