Open Salamander: Dialog Boxes

Set Speed Limit

Use this dialog box to set a speed limit for a running Copy or Move operation. You can open this dialog from the operation progress dialog using Set Speed Limit command from the drop-down menu on the Pause/Resume button.

Dialog Box Options

Do not use speed limit
When selected, indicates that no speed limit will be applied.
Set speed limit
When selected, files will be copied or moved with the specified speed limit. Set the speed limit in bytes, kilobytes, megabytes, or gigabytes per second (B/s, KB/s, MB/s, or GB/s).
