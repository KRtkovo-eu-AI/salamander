Refreshing Panel

Open Salamander automatically updates open panels. For example, when you save a new file from an application, Open Salamander displays the new filename immediately. Sometimes Open Salamander cannot complete an update when, for example, you are using network directories or you have changed floppy disks. You can update the active panel manually when this occurs.

To manually update a panel:

Menu:Choose Left/Refresh for the left panel or Right/Refresh for the right panel.
Shortcut key:Ctrl+F9 or Ctrl+R
Toolbar:Choose the Refresh button on the Top, Middle, or Bottom toolbar.
