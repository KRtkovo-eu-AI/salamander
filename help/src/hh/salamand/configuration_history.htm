Open Salamander: Dialog Boxes: Configuration

History

Specify if recently used values (history) should be stored in the registry as a part of Open Salamander and plugins configuration.

Dialog Box Options

Save history
When checked, recently used values (history) will be stored in the registry as a part of Open Salamander and plugins configuration.
Save List of Working Directories
When checked, the list of working directories (see List of Working Directories) will be stored in the registry as a part of Open Salamander configuration.
Enable Command line history
When checked, recently used commands from the Command Line will be kept in Command line history (you can access them using the button with down arrow on the right side of the Command line). If your commands contain passwords or other sensitive data, it's recommended to turn this option off.
Save Command line history
Has sense only if Enable Command line history box is checked. When checked, Command line history will be stored in the registry as a part of Open Salamander configuration.
Clear History...
Clears all history related data from memory. If you want to clear them also from the registry (configuration), you need to save configuration after using this command.
