Open Salamander: Dialog Boxes: Configuration

Editors

Allows you to associate file extensions to the external editor.

Dialog Box Options

File masks priority list
Lists all of extensions associated to the editors. More extensions separated with semi-colon (;) can be associated to the editor.
Command
Specifies path to the external program used for editing file.
Arguments
Specifies parameters passed to the external editor.
Initial directory
Specifies initial directory used for the external editor.
